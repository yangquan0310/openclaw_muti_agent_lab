# 修改文档

文档版本控制与修改管理工具。

## 功能描述

本脚本提供标准化的文档修改流程，支持：

- **版本控制**：按照语义化版本规范（major.minor.patch）管理文档版本
- **自动备份**：修改前自动备份旧版本到草稿目录
- **元数据更新**：自动更新项目元数据中的版本信息
- **修改记录**：自动记录到工作日志

## 使用方法

### 基本流程

1. 确认项目名称和文件路径
2. 提供修改后的完整内容
3. 选择版本升级方式（可选，默认patch）
4. 执行修改流程

### 版本升级规则

| 升级类型 | 说明 | 示例 |
|----------|------|------|
| major | 重大变更，不兼容修改 | v1.0.0 → v2.0.0 |
| minor | 功能新增，向后兼容 | v1.0.0 → v1.1.0 |
| patch | 问题修复，向后兼容 | v1.0.0 → v1.0.1 |

## 文件结构

```
修改文档/
```

## 依赖

- 项目已存在
- 有效的文件路径

## 输出说明

修改后的文件将保存在原路径，旧版本备份至 `草稿/{filename}/{filename}_v{版本}.md`。
