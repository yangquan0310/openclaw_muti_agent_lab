# 撰写脚本

## 触发条件
需要创建新的操作脚本

## 输入
- 脚本名称
- 脚本功能描述
- 触发场景

## 步骤

1. **确认脚本需求**
   - 明确脚本命名
   - 明确脚本功能
   - 明确触发条件

2. **撰写脚本内容**
   
   按照五要素结构撰写核心脚本文件：
   
   ```markdown
   # {脚本名}

   ## 触发条件
   [描述什么时候需要运行这个脚本]

   ## 依赖
   [列出依赖哪些其他脚本或技能]

   ## 输入
   [列出需要哪些输入参数]

   ## 步骤
   [标题采用动宾结构，明确动作链条]

   ## 输出
   [列出脚本会输出什么]
   ```

3. **撰写 README.md**
   - 脚本功能说明
   - 文件结构说明
   - 使用方法

4. **撰写 SKILL.md**
   - 技能元数据（name, description）
   - 触发条件
   - 执行方式
   - 依赖
   - 输入
   - 步骤
   - 输出

5. **检查脚本格式**
   - 检查五要素完整性
   - 检查三个文件的一致性

## 输出
- 核心脚本文件内容（{脚本名}.md）
- README.md 内容
- SKILL.md 内容
- 不涉及文件存储
