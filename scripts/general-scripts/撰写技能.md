# 撰写技能

## 触发条件
需要创建新的技能

## 输入
- 技能名称
- 技能功能描述
- 触发场景

## 步骤

1. **确认技能需求**
   - 明确技能命名
   - 明确技能功能
   - 明确触发条件

2. **撰写核心技能文件**
   
   如果是结构化技能（包含代码）：
   ```
   skills/{技能名}/
   ├── *.py              # Python脚本
   ├── *.sh              # Shell脚本
   ├── README.md         # 技能说明
   └── SKILL.md          # 技能规范
   ```
   
   如果是非结构化脚本（纯Markdown）：
   ```
   scripts/{脚本名}/
   ├── *.md              # 脚本内容
   ├── README.md         # 脚本说明
   └── SKILL.md          # 技能规范
   ```

3. **撰写 README.md**
   - 技能功能说明
   - 文件结构说明
   - 使用方法

4. **撰写 SKILL.md**
   - 技能元数据（name, description）
   - 触发条件
   - 执行方式
   - 依赖
   - 输入
   - 步骤
   - 输出

5. **检查完整性**
   - 检查技能文件结构
   - 检查三个文件的一致性

## 输出
- 核心技能文件内容
- README.md 内容
- SKILL.md 内容
- 不涉及文件存储
