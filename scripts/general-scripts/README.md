# 通用脚本

本目录包含大管家的通用脚本集合，提供文档管理、项目创建、日志记录等标准化操作流程。

## 脚本列表

| 脚本名称 | 描述 | 文件 |
|---------|------|------|
| 修改文档 | 修改已存在的文档文件，保存新版本并记录修改历史 | [修改文档.md](修改文档.md) |
| 记录工作日志 | 任务执行完成后记录工作日志 | [记录工作日志.md](记录工作日志.md) |
| 管理项目元数据 | 管理项目元数据和云文档映射关系 | [管理项目元数据.md](管理项目元数据.md) |
| 撰写脚本 | 按照六要素SOP规范撰写新脚本 | [撰写脚本.md](撰写脚本.md) |
| 检索文献 | 使用Semantic Scholar和Zotero检索文献 | [检索文献.md](检索文献.md) |

## 格式规范

所有脚本文件遵循六要素SOP结构：

```
## 触发条件
## 执行方式
## 依赖
## 输入
## 步骤
## 输出
```

## 使用说明

1. **调用脚本**: 直接调用对应的脚本文件（按脚本名称）
2. **SOP流程**: 每个脚本遵循触发条件、执行方式、依赖、输入、步骤、输出的完整结构
3. **依赖关系**: 部分脚本之间存在依赖关系，请查看脚本内容中的依赖说明
