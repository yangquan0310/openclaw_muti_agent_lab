## 功能描述

本脚本用于进行文献检索，建立项目知识库并生成检索报告。使用Semantic Scholar作为唯一允许的学术文献检索来源，支持文献分级、分类和统计分析。

## 使用方法

1. **细化检索要求**：明确研究主题、扩展关键词、构建检索词组合
2. **执行检索**：使用Semantic Scholar进行多轮检索
3. **整理文献**：排除非期刊文献，添加标签分类
4. **统计分析**：生成各类统计指标
5. **撰写报告**：生成结构化检索报告
6. **更新元数据**：同步到项目元数据

## 文献分级标准

| 级别 | 引用量 | 标记 | 处理方式 |
|------|--------|------|----------|
| 奠基 | >500 | 🔴 | 全部保留 |
| 重要 | 50-500 | 🟡 | 全部保留 |
| 一般 | <50 | 🔵 | 筛选保留 |

## 文献类型分类

| 类型 | 标记 | 判断标准 |
|------|------|----------|
| 综述 | 📖 | 标题含review/meta-analysis/systematic review |
| 理论 | 💡 | 期刊标题含perspective/opinion/commentary |
| 实证 | 📊 | 摘要含participant/subject/sample/method/result |
| 待分类 | 📋 | 无法明确归类的文献 |

## 检索报告结构

1. **检索概况**：关键词、搜索引擎、搜索过程、剔除标准、文献数量
2. **文献统计**：主题、年份、类型、重要性、主要作者、主要期刊
3. **高引用文献**：按引用数排序的前30篇（APA 7th格式）

## 禁止规则

- 严禁使用Semantic Scholar之外的任何检索方式
- 严禁编造、杜撰文献信息或检索结果
- 检索出错必须如实报告错误类型和原因
