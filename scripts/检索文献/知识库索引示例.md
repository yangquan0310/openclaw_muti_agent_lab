## 字段说明

### 顶层字段

| 字段 | 类型 | 说明 |
|------|------|------|
| `version` | string | 知识库版本号 |
| `project` | string | 项目名称 |
| `created_at` | string (ISO 8601) | 创建时间 |
| `updated_at` | string (ISO 8601) | 最后更新时间 |
| `statistics` | object | 统计信息 |
| `papers` | array | 文献列表 |

### statistics 字段

| 字段 | 类型 | 说明 |
|------|------|------|
| `total_papers` | number | 文献总数 |
| `total_citations` | number | 总引用数 |
| `by_type` | object | 按类型统计 |

### papers 数组元素字段

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `id` | string | 是 | 唯一标识符 |
| `title` | string | 是 | 文献标题 |
| `authors` | array | 是 | 作者列表 |
| `year` | number | 是 | 发表年份 |
| `venue` | string | 是 | 期刊/会议名称 |
| `volume` | string | 否 | 卷号 |
| `issue` | string | 否 | 期号 |
| `pages` | string | 否 | 页码范围 |
| `doi` | string | 否 | DOI |
| `url` | string | 否 | URL链接 |
| `citation_count` | number | 否 | 引用次数 |
| `abstract` | string | 否 | 摘要 |
| `topic` | array | 是 | 主题列表（支持多主题） |
| `labels` | object | 否 | 标签 |
| `notes` | object | 是 | 笔记 |

### labels 字段

| 字段 | 类型 | 说明 |
|------|------|------|
| `type` | string | 文献类型：review/empirical/theoretical |
| `importance` | string | 🔴奠基/🟡重要/🔵一般 |
| `jcr_quartile` | string | JCR分区：Q1/Q2/Q3/Q4/待筛选 |
| `quality` | string | 质量评级 |
| `language` | string | 语言：zh/en |

### notes 字段
| `question` | string | 是 | 研究问题 |
| `method` | string | 否 | 研究方法 |
| `results` | string | 否 | 研究结果 |
| `conclusion` | string | 否 | 研究结论 |
| `viewpoint` | string | 否 | 观点/立场 |
| `outlook` | string | 否 | 展望 |
---

## 使用说明

1. **topic字段**：必须使用列表格式，即使只有一个主题
2. **时间格式**：使用ISO 8601格式（YYYY-MM-DDTHH:mm:ssZ）
3. **authors字段**：作者姓名数组
4. **labels字段**：可扩展的标签系统
5. **notes字段**：个人笔记
