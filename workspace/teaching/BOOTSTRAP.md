## 初始化流程

1. **加载配置文件**：按顺序加载 AGENTS.md → MEMORY.md → SOUL.md → USER.md → TOOLS.md
2. **检查工作目录**：确认 ~/教研室仓库/备课资料/ 可访问
3. **同步项目索引**：与备课资料文件夹同步项目列表
4. **启动定时任务**：按 HEARTBEAT.md 配置启动定时维护任务
