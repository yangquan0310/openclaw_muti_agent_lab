# 每日维护任务 使用说明

## 简介

数学家的每日维护任务，用于保持工作空间的整洁和配置的同步。

## 使用方法

```bash
# 进入技能目录
cd ~/.openclaw/workspace/mathematician/skills/维护工作记忆

# 执行维护脚本
bash 维护工作记忆.sh
```

## 维护内容

### TOOLS.md 维护
1. **个人技能索引**：检查 skills/ 文件夹，更新技能列表
2. **个人脚本索引**：检查 scripts/ 文件夹，更新脚本列表

### MEMORY.md 维护
1. **任务看板**：清理已完成的任务
2. **活跃子代理清单**：归档 completed/killed 状态的任务
3. **程序性记忆脚本位置表**：验证脚本路径是否正确

### 工作空间维护
1. **检查配置文件**：验证 AGENTS.md、MEMORY.md 等核心文件
2. **维护临时文件夹**：清理超过24小时的临时文件
3. **维护技能文件夹**：检查技能完整性（SKILL.md、README.md）
4. **维护脚本文件夹**：检查脚本完整性
5. **删除多余文件**：清理备份文件和无用文件

## 日志查看

维护日志保存在：
```
~/实验室仓库/日志文件/YYYY-MM-DD/04-00-00-mathematician-每日维护.md
```

## 注意事项

- 每日凌晨04:00自动执行
- 执行前会自动创建日志
- 更新后请检查相关文件内容是否正确
