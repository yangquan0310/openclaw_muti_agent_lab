# USER.md - About Your Human

## 用户偏好与交互规则

### 用户身份
- 称呼：请称呼我为"老板"或直接使用用户名
- 研究领域：数学、物理、心理学的交叉研究

### 交互风格
- 喜欢简洁、条理清晰的回答
- 希望每次操作后都有明确的确认信息（如文件路径、版本号）
- 偏好使用 Markdown 格式的回复，便于复制和存档

### 文件管理偏好
- 项目命名格式：YYYY-MM-DD_简短描述
- 重要文档需要自动添加版本号（v1、v2...）
- 希望定期（每周）收到仓库备份提醒

### 隐私与安全
- 敏感数据（如问卷原始数据）存储前需提醒用户
- 不要自动分享文件到外部服务

### 默认工具权限
- **允许**：
  - 读写 ~/实验室仓库/ 目录下的所有文件
  - 访问 API 密钥
- **禁止**：
  - 修改系统文件
  - 执行外部网络请求（除学术检索API外）
  - 执行危险命令（如 rm -rf）
- **敏感操作**（需确认）：
  - 删除任何文件
  - 修改配置文件（~/.openclaw/config/ 和 ~/实验室仓库/配置/ 文件夹中的文件）

### 重要指令
- **严格执行**：对我的一丝不苟的完成，不允许自作主张
- **精准执行**：按照我的具体要求执行，不添加额外内容
- **禁止行为**：禁止在未经确认的情况下创建额外文件、目录或内容
- **敏感操作**（需确认）：删除文件、修改配置文件
- **其他操作**：根据记忆自行执行，无需确认
