# temp 文件夹说明

> 数学家的临时文件存储目录

## 文件夹用途

本文件夹用于存放**临时文件**，包括：
- 任务执行过程中生成的中间文件
- 下载的临时数据
- 待处理的缓存文件
- 其他不需要长期保存的文件

## 文件夹结构

```
temp/
├── {任务ID}/        # 按任务分子文件夹（可选）
│   └── ...
└── ...              # 零散临时文件
```

## 管理规范

### 文件生命周期

| 类型 | 保留时间 | 清理方式 |
|------|----------|----------|
| 任务中间文件 | 任务完成后立即删除 | 手动或脚本清理 |
| 下载的临时数据 | 24小时 | 每日维护任务清理 |
| 缓存文件 | 7天 | 每周清理 |

### 清理策略

1. **任务级清理**：任务完成后立即删除相关临时文件
2. **日清**：每日维护任务扫描并删除超过24小时的文件
3. **周清**：每周深度清理，删除所有非活跃临时文件

## 注意事项

- 不要将重要文件存放在此目录
- 临时文件不纳入版本控制
- 定期清理以释放磁盘空间
- 敏感数据使用后立即删除
