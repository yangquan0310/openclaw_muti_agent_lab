## 表格排序规则

### 列排序（从左到右）

| 变量名 | 说明 | 内容/值 |
|--------|------|---------|
| 执行时间 | 定时任务触发时间 | 每日 HH:MM 格式，如：每日 03:00 |
| 负责人 | 执行任务的代理名称 | 代理中文名（agentId），如：系统管理员（main） |
| 任务ID | 任务唯一标识符 | UUID格式，如：`b6a6b07d-384d-43fb-ab80-0b713e8a8289` |
| 任务名称 | 任务简短描述 | 简短描述任务内容，如：每日自动提交Git |
| 功能描述 | 任务详细功能说明 | 详细描述任务功能，如：自动提交所有更改并推送到development分支 |
| 脚本位置 | 执行任务相关脚本存储位置 | 脚本文件路径，如：`~/.openclaw/workspace/main/scripts/` |
| 执行方式 | 任务执行方式 | 子代理执行 / 主代理执行 |
| cron状态 | 定时任务状态 | `{"enabled": "已启用，任务正常执行中", "disabled": "已禁用，任务暂停执行", "error": "执行出错，任务执行失败需要检查", "pending": "待配置，任务已创建但尚未启用"}` |

### 行排序
- 按照**执行时间**增序排列（从早到晚）
- 例如：00:00 → 03:00 → 04:00 → 05:00
