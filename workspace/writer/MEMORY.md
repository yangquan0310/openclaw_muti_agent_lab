### 工作记忆使用规则

> 详细规范参见 `skills/agent_self_development/working_memory/SKILL.md`
> 本文件仅记录代理特定的实践细节

#### 代理特定规则
- 为每个任务分配唯一任务ID（格式：T001, T002, ...）
- 发现偏离计划超过 10% → 暂停并更新状态为`paused`，请求用户指示

#### 定时清理
- **执行时间**：每日 00:00（Asia/Shanghai）
- **执行方式**：主代理执行
- **任务内容**：
  1. 扫描「活跃子代理清单」
  2. 将 `completed` 状态任务归档到「事件记忆」
  3. 直接删除 `killed` 状态任务（不归档）
  4. 从清单中删除已处理的任务
  5. 记录清理日志

## 程序性记忆(Procedural Memory)

### 脚本索引

| 技能名称 | 触发示例 | 描述 | 路径 |
|---------|---------|------|------|
| 学术篇章撰写脚本 | 撰写完整学术篇章 | 撰写完整学术论文章节的脚本 | `skills/学术篇章撰写脚本/SKILL.md` |
| 学术段落撰写脚本 | 撰写学术段落 | 撰写学术论文段落的脚本 | `skills/学术段落撰写脚本/SKILL.md` |
| 学术句子撰写脚本 | 撰写学术句子 | 撰写学术论文句子的脚本 | `skills/学术句子撰写脚本/SKILL.md` |

### 条件-行动规则（If-Then Rules）

| 条件 | 行动 |
|------|------|
| 需要撰写完整学术篇章 | 调用 学术篇章撰写脚本 |
| 需要撰写学术段落 | 调用 学术段落撰写脚本 |
| 需要撰写学术句子 | 调用 学术句子撰写脚本 |
| 需要并行执行多个任务 | **最多创建2个子代理**，其余任务等待或主代理执行 |
| MD文档太大（超过5000字符） | 使用 腾讯文档分段上传工具 |
