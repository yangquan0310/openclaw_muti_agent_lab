# Memory 文件夹

> 写作助手（writer）工作日志存储目录

---

## 功能说明

本文件夹用于存储写作助手的任务执行日志和工作记录，包括：
- 每日工作日志（按日期命名）
- 详细任务执行记录（按日期归档到子目录）
- 事件记忆详细日志

---

## 文件结构

```
memory/
├── README.md                   # 本文件
├── YYYY-MM-DD.md              # 每日工作日志
├── YYYY-MM-DD/                # 详细日志目录
│   └── HH-MM-SS-{event}.md   # 事件详细日志
└── .dreams/                   # 梦境记录（如启用）
```

---

## 日志规范

- 每日工作日志：记录当天所有任务摘要
- 详细日志：每个重要事件单独成文件，包含完整上下文
- 命名格式：`HH-MM-SS-{简短事件描述}.md`
