# skills 文件夹

## 功能
存放写作助手的个人技能。

## 结构规范
每个技能是一个独立的子文件夹，包含：
```
skills/{技能名}/
├── SKILL.md      # 技能定义文件（必须）
├── *.py          # 脚本文件（可选）
└── README.md     # 功能描述（可选）
```

## 当前技能列表

| 技能名 | 描述 |
|--------|------|
| 学术句子撰写脚本 | 撰写符合学术规范的中英文句子 |
| 学术段落撰写脚本 | 撰写独立学术段落，论证特定论点 |
| 学术篇章撰写脚本 | 撰写完整学术篇章，整合段落形成连贯论述 |
