# skills 文件夹

## 功能
存放写作助手的**个人技能**（结构化程序软件）。

## 结构规范
每个技能是一个独立的子文件夹，包含：
```
skills/{技能名}/
├── *.py          # Python程序文件（可选）
├── *.sh          # Shell脚本文件（可选）
├── SKILL.md      # 技能说明文档（必须）
└── README.md     # 功能描述文档（必须）
```
