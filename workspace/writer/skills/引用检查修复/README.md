# 引用检查修复

## 功能描述
提供文献引用检查和修复功能，包括引用计数、引用映射、引用格式修复等。

## 文件列表
- check_all_refs.py - 全面引用检查
- check_refs.py - 引用检查
- citation_map.py - 引用映射
- count_citations.py - 引用计数
- detailed_check.py - 详细检查
- fix_citations.py - 引用修复
- fix_references.py - 引用修复
- fix_references_v2.py - 引用修复v2
- replace_citations.py - 引用替换

## 使用方法
各脚本可独立运行，具体使用方法参见各文件内注释。
