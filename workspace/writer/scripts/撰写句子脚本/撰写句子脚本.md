# 撰写句子脚本

## 触发条件
需要撰写符合学术规范的中英文句子

## 依赖
- 无

## 输入
- 句子主题或核心概念
- 语言类型（中文/英文）
- 句子类型（陈述/定义/论证/描述）
- 相关实体名称

## 步骤

### 1. 确定主语
- 识别核心实体名称
- 确保主语明确具体
- 避免抽象概念作为主语

### 2. 构建动词链
- 选择准确的动词
- 确保动作链条清晰
- 可用把字句或被动句

### 3. 处理定义（如适用）
- 采用"是...的..."结构
- 使用定语修饰核心名词
- 格式：是 + 定语 + 定语 + 的 + 名词

### 4. 优化表达
- 检查实体主语
- 验证动词链清晰
- 避免抽象名词
- 确保通顺自然

### 5. 验证质量
- 检查主语明确
- 验证动词准确
- 符合语言规范
- 表达简洁清晰

## 输出
- 符合规范的句子
- 句子结构分析
