# scripts 文件夹

## 功能
存放写作助手的**非结构化脚本**（Markdown格式）。

## 结构规范
每个脚本是一个独立的子文件夹，包含：
```
scripts/{脚本名}/
├── *.md          # Markdown脚本文件（可选）
├── SKILL.md      # 脚本说明文档（必须）
└── README.md     # 功能描述文档（必须）
```

## 当前脚本列表

| 脚本名 | 描述 |
|--------|------|
| 学术篇章撰写脚本 | 撰写完整学术篇章，整合段落形成连贯论述 |
| 学术段落撰写脚本 | 撰写独立学术段落，论证特定论点 |
| 学术句子撰写脚本 | 撰写符合学术规范的中英文句子 |
