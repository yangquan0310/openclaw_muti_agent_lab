# AI降重提示工程指南

> 整合句子、段落、篇章三个层级的学术写作规范
> 用于指导学生完成AI降重，确保改写后的文本符合学术规范

---

## 一、句子层级规范

### 1.1 中文句子规范

#### 核心原则
- **以实体名称作为主语**：避免使用抽象概念或被动语态开头
- **使用动词作为行动链**：强调主体的行动，保持动作连贯
- **可用把字句和被动句处理**：根据语境灵活调整句式
- **慎用抽象名词**：除非特定学术概念，否则避免使用

#### 概念定义格式
```
是 + 定语 + 定语 + 的 + 名词
```

**示例**：
- ✅ 自传体记忆是**个体对过去个人经历的**有**意识回忆**
- ✅ 数字化存储是**将信息以数字形式保存在电子设备中的**存储**方式**
- ❌ 自传体记忆涉及个人经历（缺乏"是...的..."结构）

### 1.2 英文句子规范

#### 核心原则
- 符合英文论文规范
- 不同于中文表达习惯
- 遵循学术写作惯例
- 使用恰当的学术词汇

#### 句式特点
- 主语明确，通常为具体概念或研究对象
- 动词选择准确，避免口语化表达
- 被动语态在描述研究方法时更为常见

## 二、段落层级规范

### 2.1 段落结构要求

#### 三种基本结构

**1. 总分总结构**
- 论点 → 论据支撑 → 总结升华
- 适用于：完整独立段落

**2. 总分结构**
- 论点 → 多维度论据支撑
- 适用于：系列段落中的分论点

**3. 分总结构**
- 多维度论据分析 → 归纳总结论点
- 适用于：讨论分析类段落

### 2.2 段落内容要求

#### 论点明确
- 每个段落聚焦单一核心论点
- 主题句清晰点明核心论点
- 避免一个段落包含多个不相关论点

#### 证据匹配
- 论据与论点高度相关
- 支撑充分，论证有力
- 建立论点与证据的明确联系

#### 区分事实与观点

| 类型 | 语气强度 | 常用表达 |
|------|----------|----------|
| **事实** | 强 | 研究证实、结果显示、数据表明、实验证明 |
| **观点/推测** | 弱 | 可能是、或许、推测、有待进一步验证、可能表明 |

**示例**：
- ✅ 事实：Smith等人（2020）的研究**证实**了自传体记忆的自我关联效应
- ✅ 观点：这一效应**可能**与自我图式的激活有关

### 2.3 段落格式规范

- **长度**：通常150-300字
- **引用**：按照APA 7th格式规范添加引用标注
- **过渡**：段落间使用过渡句和衔接词

## 三、篇章层级规范

### 3.1 结构对仗原则

#### 核心要求
- **研究结果章节与研究问题章节对仗**
  - 按研究问题组织文献发现
  - 而非按文献类型或作者分类
  
- **机制解释章节与研究结果章节对仗**
  - 每个研究发现对应一个机制解释
  - 解释研究发现背后的原因

#### 示例结构
```
1. 引言
2. 研究问题
   - 问题1：...
   - 问题2：...
3. 研究方法
4. 研究结果（与研究问题对仗）
   - 结果1：对应问题1
   - 结果2：对应问题2
5. 机制解释（与研究结果对仗）
   - 机制1：解释结果1
   - 机制2：解释结果2
6. 讨论
7. 结论
```

### 3.2 篇章连贯性要求

#### 逻辑连接
- 使用过渡句和衔接词强化逻辑关联
- 调整段落顺序，优化论证节奏
- 构建论点-论据-结论的完整论证链条

#### 常用衔接词

| 关系类型 | 衔接词 |
|----------|--------|
| 递进 | 此外、而且、更重要的是、进一步地 |
| 因果 | 因此、因而、由于、导致、引起 |
| 对比 | 然而、相比之下、相反、尽管如此 |
| 并列 | 同时、同样、类似地、一方面...另一方面 |
| 总结 | 综上所述、总之、由此可见、基于以上分析 |

### 3.3 内容一致性要求

- **术语统一**：专业术语使用统一，避免前后矛盾
- **风格一致**：学术表达风格保持全篇一致性
- **逻辑连贯**：各段落论点支撑篇章核心主题
- **论证闭环**：篇章整体形成完整的论证闭环

## 四、AI降重操作指南

### 4.1 降重流程

```
步骤1：理解原文核心论点
    ↓
步骤2：识别实体和动作链
    ↓
步骤3：重构句子结构
    ↓
步骤4：重组段落逻辑
    ↓
步骤5：检查篇章连贯性
    ↓
步骤6：验证学术规范
```

### 4.2 降重技巧

#### 句子层面
1. **更换主语**：将抽象概念改为具体实体
2. **调整语序**：使用把字句或被动句
3. **替换动词**：使用同义但不同的动词
4. **重组定义**：采用"是...的..."结构

#### 段落层面
1. **调整结构**：在总分总、总分、分总之间转换
2. **重组论据**：改变论据呈现顺序
3. **添加过渡**：增加衔接词和过渡句
4. **区分语气**：明确区分事实陈述和观点推测

#### 篇章层面
1. **重构框架**：按结构对仗原则重新组织
2. **优化逻辑**：调整章节顺序，强化论证链条
3. **统一术语**：确保全文术语使用一致
4. **完善衔接**：添加章节间的过渡段落

### 4.3 降重检查清单

#### 句子检查
- [ ] 主语是否为具体实体名称？
- [ ] 是否使用动词行动链？
- [ ] 概念定义是否采用"是...的..."结构？
- [ ] 是否符合目标语言规范？

#### 段落检查
- [ ] 是否聚焦单一核心论点？
- [ ] 论据是否与论点匹配？
- [ ] 事实与观点是否区分明确？
- [ ] 引用格式是否正确（APA 7th）？

#### 篇章检查
- [ ] 是否遵循结构对仗原则？
- [ ] 段落间是否有过渡衔接？
- [ ] 术语使用是否统一？
- [ ] 论证是否形成闭环？

## 五、示例对比

### 示例1：句子降重

**原文**：
> 自传体记忆的研究表明，这种记忆与自我概念密切相关。

**降重后（符合规范）**：
> 自传体记忆是**个体对过去个人经历的**有**意识回忆**，**研究证实**这种记忆形式与自我概念**存在**密切关联。

**改进点**：
1. 添加概念定义（"是...的..."结构）
2. 明确主语（自传体记忆）
3. 使用强语气词"研究证实"标记事实
4. 使用动词"存在"明确动作关系

### 示例2：段落降重

**原文**：
> 很多研究都发现数字化存储会影响记忆。有人用实验证明了这一点。也有人通过问卷调查发现了一些规律。总之，数字化存储对记忆有影响。

**降重后（符合规范）**：

> **数字化存储对自传体记忆的影响已得到多项研究的证实**。Sparrow等人（2011）的实验研究表明，当个体预期信息可被外部存储时，其对信息本身的记忆会降低，但对信息存储位置的记忆增强。这一"谷歌效应"揭示了数字化存储可能改变记忆编码策略。此外，问卷调查研究（如Ward, 2013）进一步发现，频繁使用数字设备的个体在回忆个人经历时，更倾向于依赖外部存储而非内部记忆。综上所述，**数字化存储不仅改变了记忆的内容，更重塑了记忆的认知机制**。

**改进点**：
1. 采用总分总结构
2. 明确论点（首句主题句）
3. 提供具体研究证据（Sparrow, 2011; Ward, 2013）
4. 建立论据与论点的联系（解释"谷歌效应"）
5. 使用总结句升华论点

### 示例3：篇章降重

**原文结构**：
```
1. 引言
2. 文献综述（按作者分类）
3. 研究方法
4. 结果
5. 讨论
```

**降重后（符合规范）**：
```
1. 引言
2. 研究问题
   - 问题1：数字化存储如何影响记忆编码？
   - 问题2：数字化存储如何影响记忆提取？
3. 研究方法
4. 研究结果（与研究问题对仗）
   - 结果1：数字化存储降低信息内容记忆
   - 结果2：数字化存储增强存储位置记忆
5. 机制解释（与研究结果对仗）
   - 机制1：认知卸载机制解释结果1
   - 机制2：元认知监控机制解释结果2
6. 讨论
7. 结论
```

**改进点**：
1. 采用结构对仗原则
2. 研究结果与研究问题一一对应
3. 机制解释与研究发现一一对应
4. 形成完整的论证链条

## 六、常见错误与修正

### 6.1 句子层面

| 错误类型 | 错误示例 | 修正示例 |
|----------|----------|----------|
| 抽象主语 | 研究表明... | Smith等人（2020）的研究证实... |
| 缺乏定义 | 自传体记忆很重要 | 自传体记忆是个体对过去经历的有意识回忆，在自我认同构建中具有重要作用 |
| 语气混淆 | 这证明了...（实为推测） | 这可能表明...（推测）/ 这证实了...（事实） |

### 6.2 段落层面

| 错误类型 | 错误示例 | 修正示例 |
|----------|----------|----------|
| 多论点混杂 | 一个段落讨论多个主题 | 每个段落聚焦单一论点 |
| 论据不匹配 | 论据与论点无关 | 确保论据直接支撑论点 |
| 缺乏过渡 | 段落间跳跃 | 添加过渡句和衔接词 |

### 6.3 篇章层面

| 错误类型 | 错误示例 | 修正示例 |
|----------|----------|----------|
| 结构混乱 | 按作者或时间分类 | 按研究问题和发现组织 |
| 缺乏对仗 | 结果与问题不对应 | 确保结果章节与问题章节对仗 |
| 术语不一致 | 前后使用不同术语 | 统一专业术语 |

## 七、总结

AI降重的核心目标不仅是降低重复率，更重要的是提升文本的学术规范性。通过遵循本指南中的句子、段落、篇章三个层级的规范，学生可以：

1. **提升表达清晰度**：以实体为主语，使用动词行动链
2. **增强论证逻辑性**：论点明确，证据匹配，结构对仗
3. **确保学术规范性**：符合APA格式，区分事实与观点

**最终目标**：产出高质量、低重复、符合学术规范的学术文本。
