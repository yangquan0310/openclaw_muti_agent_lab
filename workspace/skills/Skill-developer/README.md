# 撰写技能 SOP

> 版本: 1.0.0
> 更新时间: 2026-04-13
> 作者: 大管家

## 技能概述

这是创建新技能的标准操作流程，帮助你快速封装可复用的能力，支持两种类型的技能创建：

- **结构化技能**：包含可执行代码（Python/Shell等），可以直接调用执行
- **非结构化脚本**：纯Markdown描述的操作流程，作为SOP指导执行

## 功能说明

| 功能 | 描述 |
|------|------|
| 标准文件结构 | 自动生成符合OpenClaw规范的技能文件结构 |
| 双文档规范 | 同时生成AI可读的SKILL.md和人类可读的README.md |
| YAML元数据 | 统一的元数据格式，包含名称、描述、emoji、依赖等信息 |
| 多类型支持 | 支持结构化代码技能和非结构化流程脚本两种类型 |
| 模板生成 | 提供标准模板，减少重复劳动，确保规范统一 |

## 使用方法

当你需要创建新技能时，按照以下步骤操作：

1. **明确需求**：确定技能名称、功能、触发场景和类型
2. **创建结构**：按照标准目录结构创建文件夹和文件
3. **实现功能**：编写核心代码或SOP流程文档
4. **撰写文档**：完成SKILL.md和README.md的编写
5. **检查验证**：确保所有文件内容一致，符合规范要求

## 文件结构

```
撰写技能/
├── 撰写技能.md   # 核心SOP文档，详细操作步骤
├── SKILL.md      # AI可读的技能规范，结构化格式
└── README.md     # 人类可读的技能说明，本文档
```
