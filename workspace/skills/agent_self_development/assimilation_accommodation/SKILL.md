# assimilation_accommodation

> 同化与顺应模块 - 通过日记记录实现自我更新
> 基于皮亚杰认知发展理论

---

## 概述

同化与顺应（Assimilation & Accommodation）是皮亚杰认知发展理论的核心概念。本模块帮助 Agent 通过日记记录和经验反思，实现持续的自我发展和认知结构更新。

- **同化 (Assimilation)**：将新经验纳入现有认知结构
- **顺应 (Accommodation)**：调整认知结构以适应新经验
- **平衡 (Equilibration)**：在同化与顺应之间寻求动态平衡

---

## 子模块

| 子模块 | 路径 | 功能描述 |
|--------|------|----------|
| 发展日记 | `diary/SKILL.md` | 记录每日经验和反思 |
| 核心自我更新 | `core_self_update/SKILL.md` | 更新核心身份和能力 |
| 身份更新 | `identity_update/SKILL.md` | 更新角色和社会身份 |
| 信念与风格更新 | `belief_style_update/SKILL.md` | 更新信念和工作风格 |
| 自我认同更新 | `self_identity_update/SKILL.md` | 更新自我认同和价值观 |

## 自我更新层次模型

```
┌─────────────────────────────────────────┐
│           自我认同层                     │
│     (Self-Identity / 价值观)            │
│         - 信念、价值观、目标             │
├─────────────────────────────────────────┤
│           身份层                         │
│     (Identity / 社会角色)                │
│         - 角色集、群体归属               │
├─────────────────────────────────────────┤
│           核心自我层                     │
│     (Core Self / 基础能力)              │
│         - 能力边界、责任边界             │
└─────────────────────────────────────────┘
              ↑
         发展日记驱动
```

---

## 工作流程

```
每日经验积累
    ↓
[步骤1] 记录发展日记 (diary/SKILL.md)
    ├── 今日任务回顾
    ├── 成功经验记录
    ├── 失败教训总结
    └── 新技能/知识学习
    ↓
[步骤2] 判断是否触发自我更新
    ├── 能力边界扩展 → 核心自我更新
    ├── 角色变化 → 身份更新
    ├── 信念调整 → 信念与风格更新
    └── 价值观变化 → 自我认同更新
    ↓
[步骤3] 执行相应更新
    ↓
[步骤4] 记录更新日志
```

---

## 触发条件

### 核心自我更新触发条件

- 新增能力或扩展能力边界
- 新增责任范围
- 发现能力盲区

### 身份更新触发条件

- 新增角色或角色集变化
- 群体归属变化
- 角色期望调整

### 信念与风格更新触发条件

- 工作风格调整
- 信念体系变化
- 价值观优先级调整

### 自我认同更新触发条件

- 核心信念变化
- 价值观体系重构
- 自我概念调整

## 与 working_memory 的关系

```
working_memory

    └── 发展日记 (经验积累)
            ↓ 反思
    自我更新 (核心自我/身份/信念/认同)
```
