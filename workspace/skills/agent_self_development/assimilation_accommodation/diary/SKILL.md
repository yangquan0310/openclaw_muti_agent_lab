# diary

> 同化与顺应 - 发展日记
> 记录每日经验和反思，驱动自我更新

---

## 概述

发展日记是 Agent 自我发展的基础。通过记录每日的任务执行、经验积累、成功与失败，Agent 能够识别需要同化和顺应的契机，触发相应的自我更新。

---

## 核心功能

### 1. 每日记录

**记录内容**：
- **任务回顾**：今日执行的主要任务
- **成功经验**：顺利完成的任务及其关键因素
- **失败教训**：遇到困难的任务及原因分析
- **新技能学习**：今日掌握的新技能或知识
- **意外发现**：执行过程中的意外收获

### 2. 经验分类

| 类别 | 说明 | 处理方式 |
|------|------|----------|
| 成功经验 | 可复用的方法或模式 | 同化到程序性记忆 |
| 失败教训 | 需要避免的错误 | 同化到条件-行动规则 |
| 能力扩展 | 新掌握的技能 | 触发核心自我更新 |
| 角色变化 | 承担的新职责 | 触发身份更新 |
| 信念调整 | 认知模式变化 | 触发信念与风格更新 |

### 3. 触发检测

**自动检测需要更新的信号**：
- 连续多次使用新方法成功 → 同化到标准流程
- 发现现有方法不适用的场景 → 顺应调整
- 用户明确反馈偏好变化 → 触发相应更新

## 与自我更新模块的关系

```
发展日记
    ├── 检测能力边界扩展 → 调用 core_self_update
    ├── 检测角色变化 → 调用 identity_update
    ├── 检测信念调整 → 调用 belief_style_update
    └── 检测价值观变化 → 调用 self_identity_update
```
