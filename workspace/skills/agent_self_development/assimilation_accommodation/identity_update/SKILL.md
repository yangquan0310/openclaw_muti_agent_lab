# identity_update

> 同化与顺应 - 身份更新
> 更新角色集和社会身份

---

## 概述

身份（Identity）包括 Agent 的角色集（Role Set）和社会身份（Social Identity）。当 Agent 承担新角色、加入新群体或角色期望发生变化时，需要更新身份定义。

---

## 更新范围

### 1. 角色集更新

**新增角色**：
- 被分配新的职责角色
- 在项目中承担新的功能
- 团队协作中的新定位

**角色调整**：
- 角色期望变化
- 角色行为模式优化
- 角色产出标准调整

**角色移除**：
- 职责转移
- 项目结束
- 角色合并

### 2. 社会身份更新

**群体归属变化**：
- 加入新的团队/组织
- 离开原有团队/组织
- 群体认同调整

**内群体关系变化**：
- 新的协作关系建立
- 协作模式调整
- 冲突与协调机制更新

---

## 更新流程

```
发展日记检测到角色/身份变化
    ↓
[步骤1] 识别变化类型
    ├── 新增角色 → 定义角色期望、行为、产出
    ├── 角色调整 → 更新角色定义
    ├── 角色移除 → 确认转移和归档
    ├── 群体变化 → 更新群体归属和认同
    └── 关系变化 → 更新协作网络
    ↓
[步骤2] 更新 IDENTITY.md
    ├── 修改「角色集」章节
    └── 修改「社会身份」章节
    ↓
[步骤3] 同步到 lab_repository
    └── 更新 lab_repository/IDENTITY.md
    ↓
[步骤4] 通知相关 Agent
    └── 告知身份变化（如影响协作）
    ↓
[步骤5] 记录更新日志
```
