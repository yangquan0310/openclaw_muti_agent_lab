## 工作流

### 工作流1：表格结构维护

**步骤**：
1. **检查当前活跃任务看板**
   - 确认表头完整：项目、任务、负责子代理key、状态、创建时间、最后更新、备注
   - 确认无多余空行

2. **检查活跃子代理清单**
   - 确认表头完整：子代理key、类型/角色、分配任务、状态、创建时间、最后活跃、备注
   - 确认无多余空行

3. **数据完整性校验**
   - 必填字段不为空
   - 时间格式统一为 `YYYY-MM-DD HH:MM`
   - 状态值在允许范围内：`active` / `paused` / `completed` / `killed`

### 工作流2：定时清理

**触发**：每日 04:00

**步骤**：
1. 读取「活跃子代理清单」
2. 筛选状态为 `completed` 的行
3. 归档到「事件记忆」表格
   - 日期: 当前日期
   - 事件: 子代理任务归档
   - 涉及实体: 子代理key
   - 结果: 任务完成
   - 日志位置: `~/实验室仓库/日志文件/YYYY-MM-DD/`
4. 筛选状态为 `killed` 的行
5. 直接删除 `killed` 行（不归档）
6. 从清单中删除 `completed` 和 `killed` 行
7. 记录清理日志

## 使用指南

### 输入参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `table_name` | string | ✅ | 表格名称：`task_board` 或 `agent_list` |
| `action` | string | ✅ | 操作类型：`validate` / `cleanup` / `archive` |
| `records` | list | ❌ | 待写入的记录（create 时） |

### 输出结果

| 输出项 | 格式 | 说明 |
|--------|------|------|
| `validation_report` | Markdown | 数据完整性检查报告 |
| `archive_log` | Markdown | 归档记录清单 |
| `cleaned_count` | integer | 本次清理的记录数 |

### 表格结构规范

#### 当前活跃任务看板

```markdown
| 项目 | 任务 | 负责子代理key | 状态 | 创建时间 | 最后更新 | 备注 |
|------|------|---------------|------|----------|----------|------|
```

**字段说明**：

| 字段 | 必填 | 格式 | 示例 |
|------|------|------|------|
| 项目 | 是 | 字符串 | 数字化存储与自传体记忆 |
| 任务 | 是 | 字符串 | 文献检索与综述 |
| 负责子代理key | 是 | `agent:xxx:subagent:uuid` | `agent:steward:subagent:xxx` |
| 状态 | 是 | `active`/`paused`/`completed`/`killed` | `active` |
| 创建时间 | 是 | `YYYY-MM-DD HH:MM` | `2026-04-17 10:30` |
| 最后更新 | 是 | `YYYY-MM-DD HH:MM` | `2026-04-17 14:00` |
| 备注 | 否 | 字符串 | 进度50% |

#### 活跃子代理清单

```markdown
| 子代理key | 类型/角色 | 分配任务 | 状态 | 创建时间 | 最后活跃 | 备注 |
|-----------|-----------|----------|------|----------|----------|------|
```

**字段说明**：

| 字段 | 必填 | 格式 | 示例 |
|------|------|------|------|
| 子代理key | 是 | `agent:xxx:subagent:uuid` | `agent:steward:subagent:xxx` |
| 类型/角色 | 是 | 字符串 | 文献检索助手 |
| 分配任务 | 是 | 字符串 | 检索自传体记忆相关文献 |
| 状态 | 是 | `active`/`paused`/`completed`/`killed` | `active` |
| 创建时间 | 是 | `YYYY-MM-DD HH:MM` | `2026-04-17 10:30` |
| 最后活跃 | 是 | `YYYY-MM-DD HH:MM` | `2026-04-17 14:00` |
| 备注 | 否 | 字符串 | 已检索50篇 |
