# Subagent Manager

Agent 子代理管理技能包，提供「计划 → 监控 → 调节」三阶段任务执行规范。

## 功能概述

本技能包为 Agent 提供标准化的任务执行流程，确保复杂任务能够按规范完成。

### 三阶段闭环

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  计划阶段   │ → │  监控阶段   │ → │  调节阶段   │
│  (Planning) │    │ (Monitoring)│    │ (Adjusting) │
└─────────────┘    └─────────────┘    └─────────────┘
```

### 各阶段说明

| 阶段 | 触发条件 | 核心功能 |
|------|----------|----------|
| **计划阶段** | 用户提出复杂任务需求 | 需求分析、任务拆解、子代理规划 |
| **监控阶段** | 用户明确批准执行 | 执行准备、进度追踪、偏差处理 |
| **调节阶段** | 用户认可执行结果 | 经验总结、日志撰写、收尾清理 |

## 设计原则

1. **计划先行**：任何复杂任务必须先制定详细计划，获得用户确认后方可执行
2. **监控到位**：实时追踪任务进度，及时发现和处理偏差
3. **调节闭环**：任务结束后总结经验，沉淀知识，形成闭环
