# Subagent Manager

Agent 子代理管理技能包，按照「计划 → 监控 → 调节」闭环，提供详细的三阶段任务执行规范。

## 文件说明

| 文件 | 功能 | 说明 |
|------|------|------|
| `SKILL.md` | 根路由 | 技能总控文档，包含技能概述和子模块索引 |
| `计划/SKILL.md` | 子模块 | 计划阶段详细规范：需求分析、任务拆解、子代理规划 |
| `监控/SKILL.md` | 子模块 | 监控阶段详细规范：执行准备、进度追踪、偏差处理 |
| `调节/SKILL.md` | 子模块 | 调节阶段详细规范：经验总结、日志撰写、收尾清理 |
| `_meta.json` | 元数据 | 技能元数据配置 |
| `README.md` | 说明文档 | 给人类看的技能介绍 |

## 工作流

### 工作流1：计划阶段

**执行步骤**：
1. 需求分析与原则确认
2. 目标与约束梳理
3. 任务拆解与子代理规划
4. 风险与成功标准定义
5. 整合与汇报

**详细规范**：详见 `计划/SKILL.md`

---

### 工作流2：监控阶段

**执行步骤**：
1. 上下文准备
2. 传递上下文与进行子代理执行
3. 进度追踪与更新
4. 子代理结果同步
5. 结果核验与处理
6. 阶段结果汇总与反馈
7. 执行反馈

**详细规范**：详见 `监控/SKILL.md`

---

### 工作流3：调节阶段

**执行步骤**：
1. 差异与经验总结
2. 任务日志撰写
3. 工作记忆清理与归档
4. 收尾清理
5. 最终交付准备

**详细规范**：详见 `调节/SKILL.md`

---

### 调节阶段
详见：~/.openclaw/skills/subagent-manager/调节/SKILL.md
