# temp 文件夹

## 功能说明

本文件夹用于存放审稿助手的**临时工作文件**，按任务分类存储，定期自动清理。

## 文件夹结构

```
temp/
└── {任务名}/
    ├── {临时文件1}
    ├── {临时文件2}
    └── ...
```

## 使用规范

1. **按任务分类** - 每个任务创建独立子文件夹
2. **及时清理** - 任务完成后及时删除临时文件
3. **自动清理** - 系统自动删除 7 天前的临时文件

## 清理策略

- **自动清理**：每日维护任务会删除 7 天前的文件
- **手动清理**：任务完成后可立即删除
- **空目录清理**：自动删除空文件夹

## 注意事项

- 不要在此存放重要文件
- 不要在此存放敏感数据
- 定期检查和清理
