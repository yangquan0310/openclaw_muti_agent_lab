# scripts 文件夹

## 功能说明

本文件夹存放审稿助手的**非结构化脚本**，以 Markdown 格式（.md）为主，配合 SKILL.md 和 README.md 使用。

## 文件夹结构

```
scripts/
└── {脚本名}/
    ├── {脚本名}.md    # 脚本内容（Markdown格式）
    ├── SKILL.md      # 技能元数据（触发条件、输入输出等）
    └── README.md     # 使用说明文档
```

## 与 skills/ 的区别

| 特性 | scripts/ | skills/ |
|------|----------|---------|
| 文件格式 | .md | .py / .sh / .js |
| 结构类型 | 非结构化 | 结构化程序 |
| 执行方式 | 人工按步骤执行 | 自动/脚本执行 |
| 适用场景 | 复杂流程、需要人工判断 | 自动化任务、定时任务 |
