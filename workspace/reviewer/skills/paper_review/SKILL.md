# SKILL.md - paper_review

## 技能元数据

| 属性 | 值 |
|------|-----|
| **技能名称** | paper_review |
| **技能类型** | 质量审查 |
| **版本** | 1.0.0 |
| **作者** | 审稿助手 (reviewer) |
| **创建时间** | 2026-04-11 |

## 触发条件

收到论文评审请求，需要评估论文质量。

## 输入

1. **论文文件路径** - 待审查的论文文件路径（.docx 或 .md 格式）
2. **项目路径** - 项目根目录路径
3. **轮次** - 审稿轮次（如 R1、R2）
4. **检查清单路径** - 审稿论文检查清单.md 的路径

## 输出

1. **审稿意见文件** - `{项目路径}/审稿意见/{论文标题}_{轮次}.md`
2. **元数据更新** - `{项目路径}/元数据.json`

## 执行步骤

1. 读取论文文件
2. 创建审稿意见文件
3. 逐条核查检查清单
4. 撰写审稿意见
5. 更新元数据.json
6. 同步到管理元数据.md

## 文件结构

```
scripts/paper_review/
├── S1-论文审稿.md    # 审稿脚本内容
├── SKILL.md          # 技能元数据
└── README.md         # 使用说明
```

## 检查清单模块

- 标题与摘要
- 引言与文献综述
- 研究方法
- 结果呈现
- 讨论与结论
- 参考文献
- 格式规范
- 学术伦理

## 注意事项

- 需要 docx 技能读取 Word 文档
- 严格按照检查清单逐项审查
- 问题按优先级排序（高/中/低）
