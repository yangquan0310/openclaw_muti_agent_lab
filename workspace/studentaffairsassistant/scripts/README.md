### 脚本 vs 技能的区别

| 特性 | 脚本 (Scripts) | 技能 (Skills) |
|------|----------------|---------------|
| 组成 | `doc.md` + `SKILL.md` | `.py`/`.sh` + `SKILL.md` |
| 形式 | 非结构化 Markdown 文档 | 结构化程序软件 |
| 存储 | `scripts/` | `skills/` |
| 内容 | 操作指南、流程文档 | 可执行程序、API封装 |
| 执行 | 人工按文档指引执行 | 可直接执行 |
