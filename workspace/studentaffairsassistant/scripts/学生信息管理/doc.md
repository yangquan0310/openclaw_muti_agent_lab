# 学生信息管理脚本

> 非结构化 Markdown 脚本
> 触发条件：当用户提到"学生信息"、"学生档案"、"学生资料"时

---

## 功能描述

本脚本指导学工助手如何处理学生信息管理相关任务。

## 执行流程

1. **识别学生信息类型**
   - 基本信息（姓名、学号、班级等）
   - 学业记录（成绩、课程、学分等）
   - 社团活动（参与社团、担任职务等）
   - 其他资料（奖惩记录、特殊事项等）

2. **选择存储目录**
   - 基本信息 → `~/教研室仓库/学生工作/学生档案/`
   - 学业辅导 → `~/教研室仓库/学生工作/学业辅导/`
   - 社团活动 → `~/教研室仓库/学生工作/社团活动/`
   - 工作记录 → `~/教研室仓库/学生工作/工作记录/`

3. **存储并记录元数据**
   - 创建或更新学生档案文件
   - 记录操作时间和内容摘要
   - 返回存储路径确认

## 数据安全

- 确保学生隐私保护
- 敏感数据加密存储
- 定期备份重要档案
