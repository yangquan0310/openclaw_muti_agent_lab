# Temp 文件夹

> 学工助手（studentaffairsassistant）临时文件存储目录

---

## 功能说明

本文件夹用于存储学工助手的**临时文件**，包括：
- 任务执行过程中的中间文件
- 下载的临时资源
- 待处理的缓存数据

## 清理规则

- **自动清理**：每日维护任务会清理超过7天的文件
- **手动清理**：可随时删除不再需要的临时文件

## 注意事项

- 临时文件不应长期存储重要数据
- 任务完成后应及时清理相关临时文件
- 如需保留，请移动到适当的项目目录
