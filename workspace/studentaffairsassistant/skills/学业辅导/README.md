## 功能说明

本技能用于管理学生学业辅导相关工作，包括：

- 记录辅导过程
- 协调辅导资源

1. **查看技能文档**：阅读 `SKILL.md` 了解触发条件和使用方法
2. **执行辅导**：按照文档指引执行学业辅导流程
