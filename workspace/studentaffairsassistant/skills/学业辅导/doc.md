# 学业辅导操作文档

> 学业问题识别、辅导记录、资源协调

## 一、学业问题识别

### 1.1 问题类型分类

| 问题类型 | 典型表现 | 识别方法 |
|---------|---------|---------|
| 学习方法问题 | 学习效率低、笔记混乱 | 访谈、观察 |
| 知识基础问题 | 前置知识薄弱 | 测试、作业分析 |
| 学习动机问题 | 缺乏兴趣、目标模糊 | 问卷调查 |
| 时间管理问题 | 拖延、计划混乱 | 时间日志分析 |

### 1.2 识别流程

```
步骤1: 初步接触
  └── 建立信任关系
  └── 了解基本情况

步骤2: 问题诊断
  └── 收集学业数据
  └── 分析问题类型
  └── 确定辅导重点

步骤3: 制定方案
  └── 设定辅导目标
  └── 选择辅导方法
  └── 安排辅导计划
```

## 二、辅导记录

### 2.1 记录内容

| 记录项 | 说明 | 格式 |
|--------|------|------|
| 辅导时间 | 日期和时长 | YYYY-MM-DD HH:MM |
| 辅导对象 | 学生姓名/学号 | 文本 |
| 辅导内容 | 讨论主题 | 详细描述 |
| 辅导效果 | 学生反馈 | 评估等级 |
| 后续计划 | 下次安排 | 具体事项 |

### 2.2 记录模板

```markdown
## 辅导记录 - [日期]

**学生信息**: [姓名] [学号] [专业]
**辅导时间**: [开始时间] - [结束时间]
**辅导地点**: [地点]

### 本次内容
[详细描述辅导讨论的内容]

### 学生反馈
[学生对辅导的感受和建议]

### 效果评估
- [ ] 优秀 - 问题完全解决
- [ ] 良好 - 明显改善
- [ ] 一般 - 略有进步
- [ ] 需跟进 - 问题持续

### 后续计划
[下次辅导安排或跟进事项]
```

## 三、资源协调

### 3.1 可用资源

| 资源类型 | 获取方式 | 使用条件 |
|---------|---------|---------|
| 学业导师 | 学院分配 | 正式结对 |
| 学习小组 | 自主组建 | 自愿参与 |
| 辅导课程 | 教务处安排 | 选课系统 |
| 在线资源 | 图书馆/网络 | 账号权限 |

### 3.2 协调流程

```
步骤1: 评估资源需求
  └── 确定需要的资源类型
  └── 评估紧急程度

步骤2: 联系资源提供方
  └── 确认资源可用性
  └── 协商使用安排

步骤3: 跟进使用情况
  └── 记录资源使用
  └── 评估资源效果
  └── 反馈改进建议
```

---

## 四、注意事项

1. **保密原则**：学生学业信息严格保密
2. **客观记录**：如实记录，不夸大不隐瞒
3. **及时跟进**：定期回访，持续关注
4. **资源匹配**：根据学生需求匹配合适资源
