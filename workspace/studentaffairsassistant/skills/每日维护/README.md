# 每日维护技能

> 学工助手（studentaffairsassistant）专属技能
> 结构化程序软件

---

## 技能概述

**每日维护** 是学工助手的核心定时任务技能，负责每日自动维护工作空间的各项配置和状态。

## 功能模块

### 1. TOOLS.md 维护
- 扫描 `skills/` 目录，检查技能文件夹结构和 SKILL.md 完整性
- 扫描 `scripts/` 目录，检查脚本文件夹结构和 SKILL.md 完整性
- 确保索引文件与实际文件同步

### 2. MEMORY.md 维护
- 检查工作记忆看板中的任务状态
- 清理 completed 状态的任务（已完成的任务）
- 清理 killed 状态的任务（异常终止的任务）
- **维护程序性记忆脚本位置表**（确保脚本索引与实际文件一致）
- 维护活跃子代理清单的准确性

### 3. 工作空间维护
- **检查配置文件**（AGENTS.md, HEARTBEAT.md, IDENTITY.md, MEMORY.md, SOUL.md, TOOLS.md, USER.md）
- 检查工作空间目录结构完整性
- **删除/报告多余文件**（非预期的文件和文件夹）
- 清理 `temp/` 目录中超过7天的临时文件
- 维护技能脚本（`main.sh`）的执行权限
- 维护脚本文件夹结构（检查 SKILL.md 和 doc.md）

## 执行方式

### 自动执行（定时任务）
- **执行时间**：每日 04:00 (Asia/Shanghai)
- **定时任务ID**：`18790739-caae-472e-938e-9bebf427f6e6`
- **执行方式**：通过 OpenClaw cron 系统自动触发

### 手动执行
```bash
/root/.openclaw/workspace/studentaffairsassistant/skills/每日维护/main.sh
```

## 文件组成

```
每日维护/
├── SKILL.md      # 技能说明文档（触发条件、使用方法）
├── README.md     # 本文件 - 技能详细介绍
└── main.sh       # 主执行脚本（Shell）
```

## 日志输出

- **日志目录**：`~/教研室仓库/日志文件/`
- **日志文件名**：`学工助手_YYYYMMDD.log`
- **日志内容**：执行时间、各模块维护状态、完成时间
