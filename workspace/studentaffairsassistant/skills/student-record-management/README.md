# 学生档案管理技能

> 学工助手核心技能 - 学生信息管理员角色支撑

---

## 功能说明

本技能用于管理学生档案，包括创建、更新、查询、列出学生档案等操作。

## 使用方法

### 创建学生档案

```bash
# 通过AI助手调用
请创建学生档案：
- 学号：2024001
- 姓名：张三
- 性别：男
- 专业：数学与应用数学
- 年级：2024级
- 班级：1班
```

### 更新学生档案

```bash
# 通过AI助手调用
请更新学生档案（学号：2024001）：
- 联系方式：13800138000
- 邮箱：zhangsan@example.com
```

### 查询学生档案

```bash
# 通过AI助手调用
请查询学生档案：
- 学号：2024001
```

### 列出所有学生档案

```bash
# 通过AI助手调用
请列出所有学生档案
```

## 档案格式

学生档案采用Markdown格式存储，包含以下信息：

```markdown
# 学生档案 - {学号}

## 基本信息

| 项目 | 内容 |
|------|------|
| 学号 | {学号} |
| 姓名 | {姓名} |
| 性别 | {性别} |
| 专业 | {专业} |
| 年级 | {年级} |
| 班级 | {班级} |

## 联系方式

| 项目 | 内容 |
|------|------|
| 电话 | {电话} |
| 邮箱 | {邮箱} |

## 学业记录

| 学期 | 课程 | 成绩 | 备注 |
|------|------|------|------|
| {学期} | {课程} | {成绩} | {备注} |

## 社团活动

| 活动名称 | 时间 | 角色 | 备注 |
|----------|------|------|------|
| {活动名称} | {时间} | {角色} | {备注} |

---

*档案创建时间: {创建时间}*
*最后更新时间: {更新时间}*
```

## 存储位置

学生档案存储在：`~/教研室仓库/学生工作/学生档案/`
