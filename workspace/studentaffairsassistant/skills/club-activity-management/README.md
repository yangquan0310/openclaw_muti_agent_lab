# 社团活动管理技能

> 学工助手核心技能 - 社团活动组织者角色支撑

---

## 功能说明

本技能用于管理社团活动，包括活动策划、成员管理、活动记录、总结报告等。

## 使用方法

### 创建活动计划

```bash
# 通过AI助手调用
请创建社团活动计划：
- 活动名称：数学建模竞赛培训
- 社团名称：数学建模协会
- 活动类型：学术
- 活动描述：为期一周的数学建模竞赛培训
- 时间：2026-05-01 至 2026-05-07
- 地点：教学楼A301
- 预算：500元
- 负责人：张三
```

### 管理社团成员

```bash
# 通过AI助手调用
请记录社团成员：
- 社团名称：数学建模协会
- 成员：张三（会长）、李四（副会长）、王五（成员）
```

### 记录活动过程

```bash
# 通过AI助手调用
请记录活动过程：
- 计划ID：ACT-2026-001
- 实际日期：2026-05-01
- 出席人员：张三、李四、王五
- 活动内容：数学建模基础培训
- 遇到的问题：投影仪故障
- 参与者反馈：内容丰富，希望增加实践环节
```

### 生成活动总结

```bash
# 通过AI助手调用
请生成活动总结：
- 计划ID：ACT-2026-001
```

## 记录格式

社团活动记录采用Markdown格式存储，包含以下信息：

```markdown
# 社团活动记录 - {活动名称}

## 基本信息

| 项目 | 内容 |
|------|------|
| 活动名称 | {活动名称} |
| 社团名称 | {社团名称} |
| 活动类型 | {活动类型} |
| 负责人 | {负责人} |

## 活动计划

### 时间安排

| 阶段 | 时间 | 内容 |
|------|------|------|
| {阶段} | {时间} | {内容} |

### 预算

| 项目 | 金额 | 备注 |
|------|------|------|
| {项目} | {金额} | {备注} |

## 活动过程

### 出席记录

| 姓名 | 角色 | 出席状态 |
|------|------|----------|
| {姓名} | {角色} | {出席/缺席} |

### 活动记录

- **日期**: {日期}
- **地点**: {地点}
- **内容**: {活动内容}
- **问题**: {遇到的问题}

## 参与者反馈

| 姓名 | 反馈内容 | 满意度 |
|------|----------|--------|
| {姓名} | {反馈} | {满意度} |

## 活动总结

- **活动效果**: {效果总结}
- **经验教训**: {经验教训}
- **后续建议**: {建议}

---

*计划创建时间: {创建时间}*
*活动记录时间: {记录时间}*
```

## 存储位置

社团活动记录存储在：`~/教研室仓库/学生工作/社团活动/`
