# 学工助手技能索引

> 学工助手（studentaffairsassistant）个人技能存储目录
> 结构化程序软件，包含代码脚本和技能文档

---

## 功能说明

本文件夹用于存储学工助手的**结构化技能**，每个技能是一个独立的程序模块，包含可执行代码和对应的技能文档。

## 文件夹结构

```
skills/
├── README.md              # 本文件 - 技能索引
└── {技能名}/
    ├── *.py               # Python 脚本（核心功能实现）
    ├── *.sh               # Shell 脚本（命令行接口）
    ├── SKILL.md           # 技能说明文档（触发条件、使用方法）
    └── README.md          # 技能详细介绍
```

## 技能列表

| 技能名称 | 路径 | 说明 | 定时任务 |
|---------|------|------|----------|
| 每日维护 | `skills/每日维护/` | 定时执行TOOLS.md维护、MEMORY.md维护（含工作记忆清理）、工作空间维护 | ✅ 每日 04:00 |

## 使用方式

1. **查看技能文档**：阅读对应技能文件夹中的 `SKILL.md`
2. **执行技能**：运行 `.sh` 脚本或调用 `.py` 模块
3. **了解详情**：阅读 `README.md` 获取详细说明

## 与 Scripts 的区别

| 对比项 | Skills | Scripts |
|--------|--------|---------|
| 内容 | 代码脚本（.py + .sh） | Markdown 文档（.md） |
| 性质 | 结构化程序 | 非结构化脚本 |
| 执行 | 可直接执行 | 需人工按步骤执行 |
| 文档 | SKILL.md + README.md | SKILL.md + README.md |
