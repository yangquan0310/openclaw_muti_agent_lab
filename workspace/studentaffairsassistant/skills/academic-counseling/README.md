# 学业辅导记录技能

> 学工助手核心技能 - 学业辅导协调员角色支撑

---

## 功能说明

本技能用于管理学业辅导工作，包括记录学业问题、制定辅导计划、跟踪辅导效果、生成分析报告等。

## 使用方法

### 记录学业问题

```bash
# 通过AI助手调用
请记录学业问题：
- 学号：2024001
- 问题类型：学习困难
- 问题描述：高等数学成绩下滑，期中考试不及格
- 严重程度：高
```

### 制定辅导计划

```bash
# 通过AI助手调用
请制定辅导计划：
- 学号：2024001
- 辅导目标：提高高等数学成绩
- 辅导方法：一对一辅导、学习小组
- 所需资源：辅导老师、学习资料
- 时间计划：每周2次，持续4周
```

### 跟踪辅导进度

```bash
# 通过AI助手调用
请记录辅导进度：
- 记录ID：AC-2024001-001
- 辅导内容：高等数学辅导第1次
- 学生反馈：理解有提升，但还需加强练习
- 效果评估：中等
```

### 生成学业分析报告

```bash
# 通过AI助手调用
请生成学业分析报告：
- 学号：2024001
- 分析周期：2026年春季学期
```

## 记录格式

学业辅导记录采用Markdown格式存储，包含以下信息：

```markdown
# 学业辅导记录 - {学号}

## 学生信息

| 项目 | 内容 |
|------|------|
| 学号 | {学号} |
| 姓名 | {姓名} |
| 专业 | {专业} |
| 年级 | {年级} |

## 问题记录

### 问题1：{问题类型}

- **描述**: {问题描述}
- **严重程度**: {严重程度}
- **记录时间**: {记录时间}

## 辅导计划

### 计划1：{计划名称}

- **目标**: {辅导目标}
- **方法**: {辅导方法}
- **资源**: {所需资源}
- **时间计划**: {时间计划}
- **负责人**: {负责人}

## 进度记录

| 日期 | 辅导内容 | 学生反馈 | 效果评估 |
|------|----------|----------|----------|
| {日期} | {内容} | {反馈} | {评估} |

## 总结

- **辅导效果**: {效果总结}
- **后续建议**: {建议}

---

*记录创建时间: {创建时间}*
*最后更新时间: {更新时间}*
```
