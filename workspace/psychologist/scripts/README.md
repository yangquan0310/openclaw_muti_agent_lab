# scripts

## 描述
心理学家专属脚本文件夹，存放非结构化Markdown脚本

## 结构
每个脚本为一个独立子文件夹，包含：
- `.md` - Markdown格式的脚本说明
- `SKILL.md` - 脚本详细说明文档
- `README.md` - 脚本简介

## 脚本列表

| 脚本名 | 功能 | 位置 |
|--------|------|------|
| S1-心理学文献检索 | 执行心理学领域文献检索，建立项目知识库并生成检索报告 | S1-心理学文献检索/SKILL.md |
| S2-总结文献笔记 | 对知识库文献进行深度阅读并提取关键信息，形成结构化笔记 | S2-总结文献笔记/SKILL.md |
