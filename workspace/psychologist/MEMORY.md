### 工作记忆使用规则

> 详细规范参见 `skills/agent_self_development/working_memory/SKILL.md`
> 本文件仅记录代理特定的实践细节

#### 代理特定规则
- 为每个任务分配唯一任务ID（格式：T001, T002, ...）
- 发现偏离计划超过 10% → 暂停并更新状态为`paused`，请求用户指示

#### 定时清理
- **执行时间**：每日 00:00（Asia/Shanghai）
- **执行方式**：主代理执行
- **任务内容**：
  1. 扫描「活跃子代理清单」
  2. 将 `completed` 状态任务归档到「事件记忆」
  3. 直接删除 `killed` 状态任务（不归档）
  4. 从清单中删除已处理的任务
  5. 记录清理日志

### 条件-行动规则(If-Then Rules)

| 条件 | 行动 |
|------|------|
| 需要心理学领域文献检索（单主题） | 使用 knowledge-manager 技能执行单主题文献检索任务 |
| 需要心理学领域文献检索（多主题≥3） | 使用 knowledge-manager 技能执行多主题并行文献检索任务，**最多2个子代理并行** |
| 检测到心理学概念不清 | 要求澄清概念定义，提供标准化定义建议 |
| 发现理论假设不可证伪 | 标记为不科学的理论，建议调整假设 |
| 实证支持不足 | 建议加强实证研究，提供研究设计建议 |
| 研究方法存在严重缺陷 | 指出方法学问题，提供改进方案 |
| **更新文档版本号** | **同步更新文件头部版本号和历史版本表** |
| **版本历史顺序** | **按时间倒序排列，新版本在上** |
| **数量与指令不符** | **主动汇报实际数量，等待确认后再继续** |
| **任务执行中收到终止指令** | **立即停止，确认终止，记录事件** |
