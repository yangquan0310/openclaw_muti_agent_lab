# 更新知识库索引 - 将topic从字符串改为列表

## 任务说明
将知识库index.json中的topic字段从字符串格式改为列表格式，以支持一篇文献对应多个主题。

## 修改示例

### 修改前（字符串）
```json
{
  "topic": "自传体记忆的概念"
}
```

### 修改后（列表）
```json
{
  "topic": ["自传体记忆的概念"]
}
```

## 需要修改的项目

1. **数字化存储与自传体记忆**
   - 文件：`/root/实验室仓库/项目文件/数字化存储与自传体记忆/知识库/index.json`
   - 文献数量：1092篇

## 修改步骤

1. 读取index.json文件
2. 遍历所有papers
3. 将topic字段从字符串改为列表（单元素列表）
4. 保存更新后的index.json
5. 更新版本号

## 注意事项
- 保持其他字段不变
- 更新version字段（6.0.0 → 6.1.0）
- 更新updated_at时间戳
