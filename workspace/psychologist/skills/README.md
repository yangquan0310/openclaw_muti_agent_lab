# skills

## 描述
心理学家专属技能文件夹，存放结构化程序软件

## 结构
每个技能为一个独立子文件夹，包含：
- `.py` / `.sh` - 可执行程序脚本
- `SKILL.md` - 技能详细说明文档
- `README.md` - 技能简介

## 技能列表

| 技能名称 | 类型 | 描述 |
|---------|------|------|
| daily-maintenance | 定时任务 | 每日维护任务 |
| memory-maintenance | 记忆管理 | 工作记忆维护 |
| search-photo-effect | 文献检索 | 照片效应文献检索 |
| update-index-verification | 数据验证 | 索引更新与验证 |
| update-tools | 文档维护 | TOOLS.md自动更新 |
| 文献分类任务 | 文献处理 | 自动分类文献主题 |
| 文献处理工具 | 文献处理 | 去重、筛选、合并 |
| 笔记合并任务 | 笔记处理 | 按主题合并笔记 |
