# Temp 文件夹

> 教务助手临时文件存储目录
> 任务执行过程中的临时文件和中间文件

---

## 功能说明

本文件夹用于存储大管家在执行任务过程中产生的**临时文件**和**中间文件**。这些文件通常是任务执行的中间产物，不需要长期保存。

## 文件夹结构

```
temp/
├── {任务名}/
│   ├── 临时文件1          # 任务中间产物
│   ├── 临时文件2
│   └── ...
```

## 使用规范

1. **按任务创建子文件夹**：每个任务在 `temp/` 下创建独立的子文件夹
2. **及时清理**：任务完成后及时清理不再需要的临时文件
3. **定期归档**：每日维护任务会检查并清理过期的临时文件

## 示例

```
temp/
├── 文献检索_20260412/
│   ├── search_results.json
│   └── temp_index.csv
└── 数据整理_20260411/
    └── processed_data.xlsx
```

## 清理策略

- **任务完成时**：子代理应清理自己产生的临时文件
- **每日维护时**：大管家执行每日维护任务，检查并清理过期文件
- **手动清理**：发现大文件或无用文件时手动删除
