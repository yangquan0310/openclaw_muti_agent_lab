# Scripts 文件夹

> 教务助手个人脚本存储目录
> 非结构化 Markdown 脚本，按步骤执行的操作指南

---

## 功能说明

本文件夹用于存储大管家的**非结构化脚本**，每个脚本是一个 Markdown 格式的操作指南，描述按步骤执行的任务流程。

## 文件夹结构

```
scripts/
├── {脚本名}/
│   ├── *.md              # Markdown 脚本（操作步骤说明）
│   ├── SKILL.md          # 脚本触发条件和使用方法
│   └── README.md         # 脚本详细介绍
```

## 使用方式

1. **查看脚本文档**：阅读对应脚本文件夹中的 `SKILL.md` 了解触发条件
2. **执行脚本**：按照 `.md` 文件中的步骤人工执行
3. **了解详情**：阅读 `README.md` 获取详细说明

## 与 Skills 的区别

| 对比项 | Scripts | Skills |
|--------|---------|--------|
| 内容 | Markdown 文档（.md） | 代码脚本（.py + .sh） |
| 性质 | 非结构化脚本 | 结构化程序 |
| 执行 | 需人工按步骤执行 | 可直接执行 |
| 文档 | SKILL.md + README.md | SKILL.md + README.md |
