# 每日维护

物理学家专属技能 - 每日工作空间维护任务

## 功能

### 1. 维护 TOOLS.md
- 维护个人技能索引
- 维护个人脚本索引

### 2. 维护 MEMORY.md
- 维护任务看板
- 维护活跃子代理清单
- 维护程序性记忆脚本位置表

### 3. 工作空间维护
- 检查配置文件
- 维护临时文件夹
- 维护技能文件夹
- 维护脚本文件夹
- 删除多余文件

## 执行方式

```bash
bash 每日维护.sh
```

## 日志位置

~/实验室仓库/日志文件/YYYY-MM-DD/04-00-00-physicist-每日维护.md
