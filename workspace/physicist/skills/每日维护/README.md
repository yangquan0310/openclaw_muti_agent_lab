# 每日维护

物理学家每日定时维护任务脚本

## 简介

本技能用于执行物理学家工作空间的每日维护任务，确保工作空间整洁、索引文件及时更新。

## 功能

1. **维护 TOOLS.md**
   - 更新个人技能索引
   - 更新个人脚本索引
   - 维护项目表

2. **维护 MEMORY.md**
   - 维护任务看板
   - 维护活跃子代理清单
   - 维护程序性记忆脚本位置表

3. **工作空间维护**
   - 检查配置文件
   - 维护临时文件夹
   - 维护技能文件夹
   - 维护脚本文件夹
   - 删除多余文件

## 使用方法

### 手动执行

```bash
bash ~/.openclaw/workspace/physicist/skills/每日维护/每日维护.sh
```

### 定时执行

已配置为每日 04:00 自动执行（通过 OpenClaw cron 系统）

## 文件结构

```
每日维护/
├── 每日维护.sh      # 主脚本
├── SKILL.md         # 技能说明文档
└── README.md        # 本文件
```

## 注意事项

- 执行前请确保工作空间路径正确
- 建议定期查看执行日志
- 如有异常请及时处理
