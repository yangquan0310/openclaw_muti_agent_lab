# 每日维护

> 大管家（Steward）每日维护任务技能
> 执行时间：每日 04:00（Asia/Shanghai 时区）

---

## 功能说明

本技能用于执行大管家的每日维护任务，包括 TOOLS.md 维护、MEMORY.md 维护和工作空间维护。

## 触发条件

- 每日定时任务触发（04:00）
- 手动执行维护命令

## 执行步骤

### 1. 维护 TOOLS.md

- 维护个人技能索引
- 维护个人脚本索引

### 2. 维护 MEMORY.md

- 维护任务看板
- 维护活跃子代理清单
- 清理 completed/killed 任务并归档

### 3. 工作空间维护

- 删除多余文件
- 维护临时文件夹
- 维护脚本文件夹

## 使用方法

```bash
# 手动执行每日维护
bash ~/.openclaw/workspace/steward/skills/每日维护/每日维护.sh
```
