# 维护配置文件

## 触发条件
- 大管家更新了AGENTS.md，需要同步给其他代理（AGENTS.md完整替换）
- 大管家更新了工作记忆/配置文件的公共结构部分
- 需要修改标有">结构由大管家维护"的章节
- **禁止**: 复制整个文件覆盖（AGENTS.md除外）、修改标有">内容由各代理维护"的章节

## 依赖
无

## 输入
- 目标Agent名称
- 需要维护的配置文件类型
- 具体维护内容

## 步骤
1. **传递完整上下文给子代理**
   - 提供目标Agent名称和配置文件类型
   - 明确维护内容
   - 提供大管家参考配置的结构规范
   - 规定输出格式

2. **监控子代理执行任务**
   - 确定目标Agent
   - 确定配置类型
   - 读取目标Agent现有配置文件
   - 根据维护内容进行修改
   - 写入目标Agent工作空间

3. **接收和验证子代理结果**
   - 确认子代理返回修改摘要
   - 验证文件完整性

4. **记录到工作日志**

## 输出
配置文件更新完成，包含修改摘要和文件路径
