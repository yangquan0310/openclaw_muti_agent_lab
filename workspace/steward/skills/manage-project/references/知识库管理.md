# 知识库管理技能适配指南

## 目录结构约定

知识库管理技能应与 `manage-project` 技能的目录结构保持一致：

```
project/
├── 文档/              # 用户上传的文档（仅用户命令可加入/移出）
├── 手稿/              # 写作助手、审稿助手等撰写的最新md文档
├── 知识库/
│   ├── 笔记/          # 笔记输出
│   └── 综述/          # 综述输出
└── 临时数据/
    ├── 草稿/          # 备份版本、中间文件
    ├── 检索条件/       # 知识库管理检索脚本的检索条件 {检索条件}.json
    └── 笔记/          # NoteExtractor提取出来的笔记文件.md
```

## 检索条件文件

### 存放位置
- **路径**: `临时数据/检索条件/{检索条件名}.json`
- **用途**: 存放知识库管理检索脚本的检索条件

### 默认检索条件模板

```json
{
  "name": "默认检索",
  "description": "知识库默认检索条件",
  "created_at": "2026-04-22",
  "updated_at": "2026-04-22",
  "query": {
    "keywords": [],
    "tags": [],
    "date_range": {
      "start": null,
      "end": null
    },
    "sources": [],
    "file_types": [".md", ".json", ".txt"]
  },
  "filters": {
    "include_drafts": false,
    "include_notes": true,
    "include_reviews": true,
    "max_results": 100
  },
  "output": {
    "format": "json",
    "fields": ["title", "path", "created_at", "tags"],
    "sort_by": "created_at",
    "sort_order": "desc"
  }
}
```

### 字段说明

| 字段 | 值类型 | 默认值 | 说明 |
|------|--------|--------|------|
| name | str | "默认检索" | 检索条件名称 |
| description | str | "" | 检索条件描述 |
| query.keywords | list | [] | 关键词列表 |
| query.tags | list | [] | 标签过滤列表 |
| query.date_range.start | str/null | null | 起始日期（ISO格式） |
| query.date_range.end | str/null | null | 结束日期（ISO格式） |
| query.sources | list | [] | 指定来源目录 |
| query.file_types | list | [".md", ".json", ".txt"] | 文件类型过滤 |
| filters.include_drafts | bool | false | 是否包含草稿 |
| filters.include_notes | bool | true | 是否包含笔记 |
| filters.include_reviews | bool | true | 是否包含综述 |
| filters.max_results | int | 100 | 最大返回结果数 |
| output.format | str | "json" | 输出格式 |
| output.fields | list | ["title", "path", "created_at", "tags"] | 返回字段 |
| output.sort_by | str | "created_at" | 排序字段 |
| output.sort_order | str | "desc" | 排序顺序 |

## NoteExtractor 笔记文件

### 存放位置
- **路径**: `临时数据/笔记/{笔记名}.md`
- **用途**: 存放 NoteExtractor 提取出来的笔记文件

### 笔记文件格式

```markdown
# 笔记标题

> 提取来源: {源文件路径}
> 提取时间: {ISO时间戳}
> 提取工具: NoteExtractor

## 摘要

{内容摘要}

## 关键点

- {关键点1}
- {关键点2}
- ...

## 详细内容

{详细笔记内容}

## 标签

#{标签1} #{标签2}
```

## 与 manage-project 集成

### 初始化项目时创建默认检索条件

```python
from scripts.manage_project import Project
import json

project = Project("/path/to/project")

# 创建默认检索条件
default_query = {
    "name": "默认检索",
    "description": "知识库默认检索条件",
    "created_at": "2026-04-22",
    "updated_at": "2026-04-22",
    "query": {
        "keywords": [],
        "tags": [],
        "date_range": {"start": None, "end": None},
        "sources": [],
        "file_types": [".md", ".json", ".txt"]
    },
    "filters": {
        "include_drafts": False,
        "include_notes": True,
        "include_reviews": True,
        "max_results": 100
    },
    "output": {
        "format": "json",
        "fields": ["title", "path", "created_at", "tags"],
        "sort_by": "created_at",
        "sort_order": "desc"
    }
}

# 保存到临时数据/检索条件/
query_path = project.project_path / "临时数据/检索条件/默认检索.json"
query_path.parent.mkdir(parents=True, exist_ok=True)
with open(query_path, "w", encoding="utf-8") as f:
    json.dump(default_query, f, ensure_ascii=False, indent=2)
```

### 读取检索条件

```python
import json
from pathlib import Path

def load_search_query(project_path, query_name="默认检索"):
    """加载检索条件"""
    query_path = Path(project_path) / "临时数据/检索条件" / f"{query_name}.json"
    if not query_path.exists():
        return None
    with open(query_path, "r", encoding="utf-8") as f:
        return json.load(f)
```

### 保存提取的笔记

```python
def save_extracted_note(project_path, note_title, content, source):
    """保存NoteExtractor提取的笔记"""
    note_path = Path(project_path) / "临时数据/笔记" / f"{note_title}.md"
    note_path.parent.mkdir(parents=True, exist_ok=True)
    
    from datetime import datetime
    timestamp = datetime.now().isoformat()
    
    note_content = f"""# {note_title}

> 提取来源: {source}
> 提取时间: {timestamp}
> 提取工具: NoteExtractor

## 内容

{content}
"""
    
    with open(note_path, "w", encoding="utf-8") as f:
        f.write(note_content)
    
    return str(note_path)
```
