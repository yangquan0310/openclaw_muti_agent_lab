# Events - 事件记录

> 详细事件记录存储
> 由 agent-self-development 工作流维护

---

## 目录结构

```
events/
├── README.md
└── YYYY-MM-DD/
    └── HH-MM-SS-{事件名}.md    # 详细事件记录
```

---

## 事件格式

每个事件文件包含：

1. **事件标题** - 清晰描述事件
2. **时间戳** - 精确到秒
3. **事件概述** - 简要描述
4. **执行详情** - 详细记录操作步骤
5. **结果/影响** - 事件的结果和影响

---

## 与 Diary 的关系

- **Events**: 详细记录每个事件的具体操作
- **Diary**: 每日汇总，强调总结和自我发展
