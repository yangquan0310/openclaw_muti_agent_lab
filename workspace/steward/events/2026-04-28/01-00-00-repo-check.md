## 命令行接口更新

今日修复了 `Maintainer.py` 的命令行参数解析问题：

### 新的命令结构

```bash
# 整理项目（需要显式指定 organize 子命令）
python3 Maintainer.py organize <项目路径>

# 整理所有项目
python3 Maintainer.py organize --all

# 归档文件
python3 Maintainer.py archive <项目路径> <文件> [--title <标题>] [--version <版本>]

# 移动文件
python3 Maintainer.py move <项目路径> <文件> <目标目录> [--new-name <新名>]

# 元数据管理
python3 Maintainer.py meta <项目路径> [--title <标题>] [--status <状态>]
```

### 子命令列表

| 子命令 | 功能 | 示例 |
|--------|------|------|
| `organize` | 整理项目文件 | `organize <项目路径>` |
| `archive` | 归档文件到草稿 | `archive <项目路径> <文件>` |
| `move` | 移动文件到标准目录 | `move <项目路径> <文件> <目标目录>` |
| `meta` | 元数据管理 | `meta <项目路径> --title "新标题"` |
