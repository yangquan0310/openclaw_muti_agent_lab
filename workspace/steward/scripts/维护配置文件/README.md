# 维护配置文件

> 大管家（Steward）脚本
> 用于维护其他Agent的配置文件

---

## 功能说明

本脚本用于同步大管家更新的公共结构部分到其他Agent的配置文件中，确保所有Agent的配置文件结构保持一致。

## 文件结构

```
维护配置文件/
├── 维护配置文件.md    # 脚本详细说明
├── SKILL.md           # 技能规范文档
└── README.md          # 本文件
```

## 使用方法

1. 查看 SKILL.md 了解触发条件和使用方法
2. 按照 维护配置文件.md 中的步骤执行

## 输入

- 目标Agent名称
- 配置文件类型（SOUL.md/IDENTITY.md/MEMORY.md等）
- 具体维护内容

## 输出

- 目标Agent的配置文件已更新
- 修改摘要
