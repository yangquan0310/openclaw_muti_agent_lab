## 使用说明

1. **调用脚本**: 直接调用对应的脚本文件（按脚本名称）
2. **综合使用**: 对于文档管理全流程，建议使用"文档全流程管理"脚本
3. **特定功能**: 如需特定功能，可单独调用对应的独立脚本
4. **SOP流程**: 每个脚本遵循触发条件、执行方式、依赖、输入、输出、步骤的完整结构
5. **依赖关系**: 部分脚本之间存在依赖关系，请查看脚本内容中的依赖说明
6. **命名规范**: 所有脚本按名称命名，便于识别和调用
