# 创建项目元数据脚本

## 触发条件
需要创建新项目时，在项目初始化完成后创建 `元数据.json`

## 执行方式
当前会话执行，根据用户提供的项目信息生成并写入元数据.json

## 输入
- 项目ID：项目名称
- 项目标题：项目标题
- 创建日期：创建日期（YYYY-MM-DD）
- 项目描述：项目描述
- 状态：active / completed

## 输出
- 创建完成的 `元数据.json` 文件路径
- 输出元数据内容确认

## 依赖
- 无（直接创建，不需要依赖其他脚本）

## 步骤

1. **确定项目路径**
   - 项目路径：`~/实验室仓库/项目文件/YYYY-MM-DD_项目名/`
   - 检查路径是否存在，如果不存在则提示错误

2. **初始化元数据结构**
   ```json
   {
     "project_id": "项目名称",
     "title": "项目标题",
     "created_date": "创建日期",
     "status": "active",
     "version": "v1",
     "description": "项目描述",
     "directories": {},
     "files": [],
     "tags": [],
     "tasks": [],
     "cloud_doc_mappings": []
   }
   ```

3. **写入元数据.json**
   - 写入到项目根目录
   - 设置正确文件权限：644

4. **验证创建**
   - 检查文件是否存在
   - 验证JSON格式是否正确

5. **记录到工作日志**
   - 记录创建时间、项目名、路径
   - 完成
