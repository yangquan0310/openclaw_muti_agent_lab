# 检索文献SOP

## 触发条件
需要根据关键词检索学术文献，并将检索结果存入项目知识库的 `index.json`

## 执行方式
创建子代理执行文献检索，主代理更新知识库

## 输入
- 项目名称：目标项目（如 `2026-04-01_数字化存储与自传体记忆`）
- 关键词：检索关键词/主题，可以多个关键词用 OR 分隔
- 最大数量：需要检索的最大文献数量（默认 100）
- 标签：可选，为检索到的文献添加标签

## 输出
- 检索结果更新到 `~/实验室仓库/项目文件/{项目名}/知识库/index.json`
- 输出检索统计：检索到 N 条，新增 M 条，更新 K 条

## 依赖
- `semantic-scholar-mcp` 技能：Semantic Scholar 文献检索
- `zotero` 技能：Zotero 本地文献库检索
- `管理知识库.md`：管理项目知识库 index.json

## 步骤

1. **确认检索需求**
   - 确认目标项目
   - 确认检索关键词
   - 确认最大检索数量
   - 确认标签

2. **定位项目知识库**
   - 检查项目目录是否存在
   - 检查 `index.json` 是否存在，不存在则创建

3. **多源文献检索**
   - **Semantic Scholar**: 使用 API 按关键词检索文献，获取标题、作者、年份、DOI、摘要、引用量
   - **Zotero**: 在本地 Zotero 库中按关键词检索已有文献
   - 合并检索结果，去重（基于 DOI）

4. **去重与整理**
   - 检查 `index.json` 中已存在的文献（DOI 比对）
   - 区分新增文献和已有文献
   - 为新增文献整理格式，添加时间戳

5. **更新项目知识库**
   - 调用 `管理知识库` 添加新增文献
   - 更新根级别 `updated_at` 和 `total_count`
   - 验证保存结果

6. **统计结果**
   - 总检索数量
   - 新增数量
   - 更新数量
   - 已存在数量

7. **记录到工作日志**
   - 记录检索时间、项目、关键词、统计结果

8. **更新 TOOLS.md 索引**
   - 在大管家 `TOOLS.md` 公共技能索引中更新文献检索技能信息
