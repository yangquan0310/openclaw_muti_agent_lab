# 通用脚本技能

提供大管家的标准化操作脚本集合，支持文档管理、项目管理、日志记录等全生命周期操作。

## 功能特性

- ✅ **文档全流程管理**：集成7个脚本功能的完整文档生命周期管理
- ✅ **标准化脚本格式**：所有脚本遵循统一的触发条件、执行方式、步骤结构
- ✅ **智能决策执行**：根据具体场景自动选择执行路径
- ✅ **SOP流程规范**：严格遵循标准操作流程规范
- ✅ **多平台支持**：支持飞书、腾讯云文档等多种云平台

## 详细说明

### 1. 文档全流程管理脚本

**功能概述**：
整合了文档管理的完整生命周期，涵盖从本地文件创建到云文档同步、版本控制的全部流程。

**核心功能**：
1. **本地文件操作**：创建、修改本地Markdown文件
2. **云文档同步**：智能上传到飞书/腾讯云文档
3. **映射管理**：维护本地文件与云文档的映射关系
4. **版本控制**：规范的版本管理和历史回溯

**使用场景**：
- 需要创建新的文档或项目文件
- 需要修改现有文档内容
- 需要将文档同步到云平台
- 需要更新文档版本
- 需要管理云文档映射关系

### 2. 创建工作日志脚本

**功能概述**：
标准化的日志记录流程，确保任务执行的完整追溯。

**使用场景**：
- 任务执行完成后记录执行情况
- 记录任务参与者和分工说明
- 保存生成文件的存储路径
- 记录任务执行过程中的问题与解决方案

**输出格式**：
```
路径: ~/实验室仓库/日志文件/YYYY-MM-DD/HH-MM-SS-[agents]-[任务类型].md
```

### 3. 创建新项目脚本

**功能概述**：
标准化的项目初始化流程，确保项目结构规范统一。

**使用场景**：
- 初始化新的研究项目
- 创建项目目录结构
- 生成项目元数据文件
- 创建项目说明文档

**目录结构**：
```
YYYY-MM-DD_项目名/
├── 原始文档/           # 用户上传文档
├── 原始数据/           # 原始数据
├── 草稿/               # 论文草稿
├── 终稿/               # 终稿
├── 模板/               # 项目相关模板
├── 知识库/             # 项目专属知识库
├── 元数据.json         # 项目元数据
└── README.md           # 项目说明
```

### 4. 查询规范脚本

**功能概述**：
标准化的规范查阅流程，确保操作符合实验室规章制度。

**使用场景**：
- 需要查阅记忆管理规范
- 需要查阅本地文件管理规范
- 需要查阅云文档管理规范
- 需要查阅日志管理规范
- 需要查阅更新管理规范

**支持的规范**：
- 记忆管理规范.md
- 本地文件管理规范.md
- 云文档管理规范.md
- 日志管理规范.md
- 更新管理规范.md

## 使用方式

### 基本使用

1. **调用脚本**: 直接调用对应的脚本文件
2. **综合使用**: 对于文档管理全流程，建议使用"文档全流程管理"脚本
3. **特定功能**: 如需特定功能，可单独调用对应的独立脚本
4. **SOP流程**: 每个脚本遵循触发条件、执行方式、依赖、输入、输出、步骤的完整结构
