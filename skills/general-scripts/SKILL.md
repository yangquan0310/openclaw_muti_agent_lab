# 通用脚本技能

提供大管家的标准化操作脚本集合，支持文档管理、项目管理、日志记录等全生命周期操作。

## 功能特性

- ✅ **模块化拆分**：每个操作一个独立脚本，按需调用
- ✅ **标准化脚本格式**：所有脚本遵循统一的六要素SOP结构（触发条件、执行方式、依赖、输入、步骤、输出）
- ✅ **智能决策执行**：根据具体场景自动选择执行路径
- ✅ **SOP流程规范**：严格遵循标准操作流程规范

## 脚本列表

| 脚本名称 | 文件名 | 描述 | 用途 |
|---------|--------|------|------|
| 修改文档 | `修改文档.md` | 修改已存在的文档文件，保存新版本并记录修改历史 | 文档修改与版本控制 |
| 记录工作日志 | `记录工作日志.md` | 任务执行完成后记录工作日志 | 任务执行记录 |
| 管理项目元数据 | `管理项目元数据.md` | 管理项目元数据和云文档映射关系 | 项目元数据管理 |
| 撰写脚本 | `撰写脚本.md` | 按照六要素SOP规范撰写新脚本 | 创建新的通用脚本 |
| 检索文献 | `检索文献.md` | 使用Semantic Scholar和Zotero检索文献并更新项目知识库 | 文献检索与入库 |

## 脚本格式规范

所有脚本文件遵循六要素SOP结构：

```markdown
## 触发条件
## 执行方式
## 依赖
## 输入
## 步骤
## 输出
```

## 使用方式

### 基本使用

1. **调用脚本**: 直接调用对应的脚本文件
2. **独立使用**: 每个脚本独立完成特定功能
3. **SOP流程**: 每个脚本遵循六要素完整结构

## 文件结构

```
general-scripts/
├── SKILL.md              # 本文件（技能说明）
├── README.md             # 脚本索引说明
├── 修改文档.md           # 修改文档并版本控制
├── 记录工作日志.md       # 工作日志记录脚本
├── 管理项目元数据.md     # 项目元数据和云文档映射管理
├── 撰写脚本.md           # 按照六要素SOP规范撰写新脚本
└── 检索文献.md           # 文献检索与知识库更新
```
