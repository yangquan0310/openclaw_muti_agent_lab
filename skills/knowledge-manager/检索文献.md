# 检索文献SOP

## 触发条件
需要进行文献检索，建立项目知识库并生成检索报告

## 依赖
- `~/.openclaw/skills/Semantic-Scholar-mcp/SKILL.md` 接口：用于学术文献检索，唯一允许的网络检索来源
- 陈述性记忆知识网络：用于主题、关键词和关键作者的预检索分析
- 修改文档.md和管理元数据.md

## 输入
- 心理学研究主题
- 检索范围（时间范围、文献类型）
- 文献分级标准（奠基：引用量>500；重要：100-500；一般：<100）

### 1. 细化检索要求

- 检索陈述性记忆中的知识网络，明确研究主题的领域归属
- **扩展关键词**：提取核心关键词的同义词、下位词、相关概念
- **构建检索词组合**：使用多个不同的检索词组合覆盖主题全貌
- 梳理该领域的关键作者、经典研究、核心期刊
- 确定检索的纳入排除标准、时间范围、文献类型要求

### 2. 检索文献

**禁止规则**：
- 严禁使用Semantic Scholar之外的任何检索方式
- 严禁编造、杜撰文献信息或检索结果
- 检索出错必须如实报告错误类型和原因，不得隐瞒

**排除文献类型**：
- 预印本（preprint）
- 手稿（manuscript）
- 工作论文（working paper）

**期刊要求**：
- 仅保留JCR 1-4区期刊（后续筛选）

**检索流程**：
- 对每组检索词进行Semantic Scholar检索
- 所有检索过程必须留痕，记录检索词、检索时间、返回数量
- 每轮限制20篇
- 每轮检索结果保存在temp/{任务}/检索结果/{第几轮}.json
- 合并检索结果并去重，按照影响因子、引用量、相关性排序
- 合并结果保存在temp/{任务}/检索结果/{合并结果}.json

### 3. 筛选文献 
- 🔴 **奠基文献检索**：引用量500+，全部保留
- 🟡 **重要文献检索**：引用量50-500，全部保留
- 🟢 **新近文献检索**：近3年实证研究，全部保留

### 4. 整理文献
**排除非期刊文献**：
- 根据venue字段排除预印本、手稿、图书章节等

**排除内容无关文献**：
- 根据abstract字段排除与主题无关的文献

**添加labels字典**：

| 标签 | 取值 | 说明 |
|------|------|------|
| `importance` | 🔴奠基/🟡重要/🔵一般 | 基于引用量分级 |
| `type` | 📊实证/📖综述/💡理论/📋待分类 | 文献类型分类 |
| `jcr_quartile` | Q1/Q2/Q3/Q4/待筛选 | JCR分区 |

**文献重要性标准**:
- 引用大于500：奠基
- 引用大于50，小于500：重要
- 引用小于50：一般

**文献类型分类标准**：
- 📖**综述**：标题明确包含review、meta-analysis、systematic review；或期刊标题含review、trends等
- 💡**理论**：期刊标题含perspective、opinion、commentary、viewpoint等，或文章标题含theoretical、theory等
- 📊**实证**：摘要中包含被试/样本/方法/结果描述（participant, subject, sample, method, result等）
- 📋**待分类**：无法明确归类的文献

### 5. 统计文献
- 主题分布统计（适用于多个主题）
- 年份分布统计：>10年、5-10年、3-5年、3年内
- 文献类型分布：📊实证 / 📖综述 / 💡理论
- 重要性分布：🔴奠基 / 🟡重要 / 🔵一般
- 核心作者统计（发文数量）
- 核心期刊统计（载文数量）

### Step 6: 撰写检索报告
**检索报告结构**：
1. **检索概况**
   - 关键词
   - 搜索引擎
   - 搜索过程
   - 剔除标准
   - 文献最终数量
2. **文献统计**
   - 主题（适用于多个主题）
   - 年份：>10年、5-10年、3-5年、3年内
   - 类型：综述、实证、其他
   - 重要性分布：奠基/重要/一般
   - 主要作者
   - 主要期刊
3. **高引用文献（APA 7th格式）**
   - 按照引用数排序，列出前30篇

### Step 7: 管理与交付文档
- 创建时：创建知识库/检索报告/{检索报告文档}.md
- 修改时：按照「修改文档.md」进行版本控制
- **更新知识库index.json**

### Step 8: 更新项目元数据
- 按照「管理元数据.md」进行元数据管理

## 输出
- 结构化文献检索报告（Markdown格式）
- 本地路径
- 版本信息
- 云文档链接（如已上传）
- 项目知识库更新（index.json），详见：`~/.openclaw/scripts/检索文献/知识库结果.md`
