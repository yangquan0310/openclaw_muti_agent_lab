## 字段说明

### 顶层字段

| 字段 | 类型 | 说明 |
|------|------|------|
| `version` | string | 知识库版本号 |
| `project` | string | 项目名称 |
| `created_at` | string (ISO 8601) | 创建时间 |
| `updated_at` | string (ISO 8601) | 最后更新时间 |
| `statistics` | object | 统计信息 |
| `papers` | array | 文献列表（list，不是dict） |

### statistics 字段

| 字段 | 类型 | 说明 |
|------|------|------|
| `total_count` | number | 文献总数 |
| `foundation_count` | number | 奠基文献数 |
| `important_count` | number | 重要文献数 |
| `general_count` | number | 一般文献数 |
| `empirical_count` | number | 实证研究数 |
| `review_count` | number | 综述研究数 |
| `theory_count` | number | 理论研究数 |

### papers 数组元素字段（按固定顺序）

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `id` | string | 是 | 唯一标识符（如paper_001） |
| `paperId` | string | 是 | Semantic Scholar ID |
| `authors` | array | 是 | 作者列表 |
| `year` | number | 是 | 发表年份 |
| `title` | string | 是 | 文献标题 |
| `venue` | string | 是 | 期刊/会议名称 |
| `volume` | string | 否 | 卷号 |
| `issue` | string | 否 | 期号 |
| `pages` | string | 否 | 页码范围 |
| `doi` | string | 否 | DOI |
| `url` | string | 否 | URL链接 |
| `abstract` | string | 否 | 摘要 |
| `topic` | array | 是 | 主题列表（支持多主题） |
| `citationCount` | number | 否 | 引用次数 |
| `labels` | object | 否 | 标签 |
| `notes` | object | 是 | 笔记 |

⚠️ **重要：papers是list，不是dict！**

### labels 字段

| 字段 | 类型 | 说明 |
|------|------|------|
| `type` | string | 文献类型：📊实证/📖综述/💡理论 |
| `importance` | string | 重要性：🔴奠基/🟡重要/🔵一般 |
| `confidence` | number | 置信度（0-1） |

### notes 字段
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `研究问题` | string | 是 | 研究问题 |
| `研究方法` | string | 否 | 研究方法 |
| `研究结果` | string | 否 | 研究结果 |
| `研究结论` | string | 否 | 研究结论 |
| `研究观点` | string | 否 | 观点/立场 |

---

## 使用说明

1. **papers字段**：必须使用数组（list）格式，不是对象（dict）
2. **字段顺序**：必须按上述固定顺序排列文献字段
3. **topic字段**：必须使用列表格式，即使只有一个主题
4. **时间格式**：使用ISO 8601格式（YYYY-MM-DDTHH:mm:ssZ）
5. **authors字段**：作者姓名数组
