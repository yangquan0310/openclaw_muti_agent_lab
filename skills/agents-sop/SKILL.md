# agents-sop

> Agent SOP（标准操作流程）技能包
> 提供详细的三阶段任务执行规范：计划阶段、监控阶段、调节阶段

---

## 技能列表

| 脚本名称 | 触发条件 | 描述 |
|---------|---------|------|
| 计划阶段.md | 用户提出复杂任务需求 | 制定执行计划的详细规范 |
| 监控阶段.md | 用户批准执行计划 | 监控子代理执行的详细规范 |
| 调节阶段.md | 任务执行完成 | 任务收尾和经验总结的详细规范 |

---

## 使用方式

在 AGENTS.md 中，三阶段速查部分引用这些脚本：

```
## 三阶段速查

### 计划阶段
详见：~/.openclaw/skills/agents-sop/计划阶段.md

### 监控阶段  
详见：~/.openclaw/skills/agents-sop/监控阶段.md

### 调节阶段
详见：~/.openclaw/skills/agents-sop/调节阶段.md
```

---

## 钩子机制

AGENTS.md 中保留核心规则和速查，详细执行步骤在这些脚本中。
确保每次任务都能按三阶段处理，同时保持 AGENTS.md 精简。
