# agent_self_development

> Agent 自我发展技能包
> 基于皮亚杰认知发展理论 + Baddeley 工作记忆模型构建的 Agent 自我进化系统

---

## 简介

本技能包提供完整的 Agent 自我发展框架，帮助 Agent 通过持续学习和反思实现自我进化。

### 理论基础

- **皮亚杰认知发展理论**：同化与顺应机制驱动认知结构进化
- **Baddeley 工作记忆模型**：管理当前活跃任务和认知过程
- **元认知理论**：计划、监控、调节三阶段闭环

---

## 核心工作流

### 工作流1：六阶段任务生命周期

```
阶段0：会话初始化
    ↓
阶段1：任务决策（快速查询/Cron → 直接执行）
    ↓
阶段2：计划（Planning）
    ↓
阶段3：执行与监控（Monitoring）
    ↓
阶段4：调节（Regulation）（条件触发）
    ↓
阶段5：任务完成与归档
```

### 工作流2：每日自我更新（代理发展核心）

```
每日 00:00 定时触发
    ↓
1. 阅读当日事件记忆
    ↓
2. 撰写/完善发展日记
    ↓
3. 阅读核心自我（MEMORY.md/SOUL.md/IDENTITY.md/skills/README.md）
    ↓
4. 同化与顺应分析
    ↓
5. 检测更新触发信号
    ↓
6. 执行相应更新（核心自我/身份/风格-信念/技能）
    ↓
7. 记录更新日志
```

## 代理发展机制

### 发展循环

```
执行任务 → 记录日记 → 同化顺应分析 → 自我更新 → 能力进化 → 更好地执行任务
     ↑                                                                    ↓
     └──────────────────────── 持续循环 ←─────────────────────────────────┘
```

### 更新类型

| 更新维度 | 触发条件 | 更新内容 | 目标文件 |
|----------|----------|----------|----------|
| **核心自我** | 能力边界变化 | 核心身份、能力边界 | MEMORY.md |
| **身份** | 角色变化 | 角色集、社会身份、协作网络 | IDENTITY.md |
| **风格-信念** | 价值观/工作方式变化 | 工作信念、交互/文档/代码/执行风格 | SOUL.md |
| **技能** | 习得/细化/淘汰 | 技能体系、技能索引 | skills/README.md |

## 定时任务配置

### 每日自我更新任务

| 属性 | 配置 |
|------|------|
| **执行时间** | `0 0 * * *`（每日 00:00，Asia/Shanghai） |
| **执行方式** | 主代理执行 |
| **触发消息** | `[cron:每日自我更新]` |
| **详细规范** | `assimilation_accommodation/SKILL.md` |

### 定时清理任务

| 属性 | 配置 |
|------|------|
| **执行时间** | `0 0 * * *`（每日 00:00，作为自我更新的一部分） |
| **执行方式** | 主代理执行 |
| **详细规范** | `working_memory/SKILL.md` |

## 使用指南

### 快速开始

1. **阅读技能文档**：
   - `metacognition/planning/SKILL.md` - 学习如何制定计划
   - `metacognition/monitoring/SKILL.md` - 学习如何监控执行
   - `assimilation_accommodation/diary/SKILL.md` - 学习如何记录日记

2. **执行每日自我更新**：
   - 等待每日 00:00 定时触发
   - 或手动发送 `[cron:每日自我更新]` 消息

3. **查看发展记录**：
   - 检查 `memory/YYYY-MM-DD/` 目录
   - 阅读日记和事件记忆

### 自定义发展

如需调整发展节奏或内容，可修改：
- `HEARTBEAT.md` - 调整定时任务配置
- 各子模块 SKILL.md - 调整具体工作流步骤
